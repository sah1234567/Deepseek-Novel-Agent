# 仙侠写作规范

## 目标

确保仙侠/修真世界观内部自洽——境界体系不跳级、战斗描写遵守跨境界碾压规则、
修炼资源有明确来源、突破有完整链条（外因→内因→质变→外界反应），
不出现战力崩坏或设定矛盾。

## 建议创建的知识库文件

单独使用本 skill 时建议: 战力系统.md, 功法技能.md, 场景追踪.md, 道具追踪.md。
视剧情: 势力追踪.md（有宗门斗争时）、地理.md（非现实地理时）。

## 境界体系刚性约束

- 境界树必须在战力系统.md 中明确定义，每个境界标注: 寿元/战力上限/突破条件
- 突破必须包含完整链条: 外因→内因→突破方式→质变(新能力/体征)→外界反应
- 禁止跳级。需要跨级突破时必须有提前≥10 章埋设的伏笔支撑
- 境界名称设定后禁止改名或换说法，别名需记录在 aliases 中
- 同一章内最多一人突破（群像可放宽至二人），突破描写必须与前文修炼进度匹配

> 例：炼气期突破到筑基期 → 外因(寻得筑基丹)→内因(炼气圆满)→突破方式(闭关服丹)→质变(凝聚筑基台，寿元+200)→外界反应(天地异象引来临近修士窥探)

## 战斗描写约束

- 同境界: 胜负取决于功法克制/战斗经验/临场发挥/环境利用
- 跨境界: 高境界必须有碾压优势；低境界获胜必须付出≥等价代价（重伤/折寿/透支）
- 每场战斗必须推动三条线中≥一条: 情节推进/角色成长/信息揭示
- 战斗中展示的新能力必须在之前的修炼描写中有铺垫
- 招式命名不可重复，同一招式多次出现需保持效果一致

> 例：筑基中期主角对战筑基圆满反派 → 主角依靠刚获得的剑意传承(有伏笔)和环境掩护(雷暴天气)险胜，但经脉受损需休养三章

## 修炼与资源约束

- 每个境界的修炼资源（灵气浓度/丹药/功法）需提前在战力系统.md 设定
- 修炼速度受灵根等级+体质+机缘三维约束，单一维度的极端值需有代价
- 资源获取必须有明确来源（师门/交易/奇遇/掠夺），同类型来源不能连续使用≥3次
- 天生异禀/血脉优势需设定限制（如血脉觉醒有冷却期/消耗寿命）

> 例：主角天灵根(灵根顶级)但体质孱弱(体质短板)，导致吸收灵气极快但肉身承载不足，需要通过炼体或寻找强化体质的丹药来平衡

## 常见错误

以下错误会导致战力崩坏或设定矛盾，必须避免：

- **战力通货膨胀**: 每个新卷的境界天花板被不断打破，前期顶尖高手后期变杂兵 → 境界树需前期全局设定，新增境界需≥15 章伏笔
- **突破即无敌**: 主角每次突破后碾压同级，无适应期 → 突破后需≥2 章的磨合期（新力量不熟悉/身体负荷/外界觊觎）
- **丹药万能化**: 任何困境靠一颗丹药解决 → 丹药有副作用/抗药性/品阶上限，同类型丹药不能连续使用≥3 次
- **功法即战力**: 获得功法=立即掌握=无敌 → 功法习得需≥3 章学习/磨合/试错，熟练度用入门→小成→大成→圆满追踪

## 与其他 skill 的交互

- 本 skill 定义修炼体系的基准规则。当与特效流派 skill（如系统/重生）叠加时，修炼速度和机缘获取可适度加快，但境界树定义和突破链条不可简化
- 本 skill 的战力系统与武侠类 skill（wuxia）互斥——不可叠加。如混合元素，选择占比≥70%的一方作为主 skill
- 当爽文类 skill（power-fantasy）激活时，突破频率和机缘密度可适度提高，但境界树结构、跨境界碾压规则、丹药副作用不可取消
