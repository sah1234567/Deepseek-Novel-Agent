# 短剧改编写作规范

## 目标

确保小说适应短剧改编需求——对话驱动情节（≥50%）、场景精简且视觉化、
章末钩子类型轮替、内心独白极少、每章至少一个可拍摄记忆点。

## 建议创建的知识库文件

与具体流派 skill 相同，无额外文件需求。可选加强: 场景追踪.md（场景复用率高，每次描写需视觉一致）。

## 叙事风格约束

- 对话驱动: 每章对话占比≥50%。用对话推进情节和展现人物性格，避免大段叙述
- 内心独白极少: 每次内心独白≤3 句，优先用动作和对话暗示内心活动
- 场景描写极简: 每个新场景≤3 句概述，聚焦"可拍摄"的视觉元素（光/色/动作/道具）
- 每章至少 1 个"可拍摄的记忆点"——一个在短剧中只需一个镜头就能传达的瞬间

> 例：不使用内心独白"她感到无比愤怒和羞辱" → 使用"她将茶杯重重磕在桌上，茶水溅出来。她盯着对方，嘴角挂着一个淡到几乎看不出的冷笑。"(1个动作+1个表情=可拍摄记忆点)

## 章末钩子约束

- 每章结尾必须有钩子，类型轮替: 悬念(信息差)/反转(认知颠覆)/冲突升级/新线索
- 钩子不能廉价化: 每 5 章中至少 1 个钩子在下一章被证伪或产生意外后果
- 连续 3 章同一类型的钩子 → 节奏警告
- 章末钩子必须在本章情节中有铺垫，不能是从天而降的意外

## 角色约束

- 核心角色数量≤常规同类型小说: 减少群像，每场景出场人数≤4 人
- 每个角色必须有≥1 个外在特征（习惯性动作/口头禅/视觉标志），方便短剧呈现
- 角色的情感变化通过可拍摄的动作表达: 摔门/沉默/抿嘴/避开视线

## 常见错误

以下错误会导致短剧改编困难，必须避免：

- **对话膨胀**: 对话占比高但全是在交换信息无戏剧张力 → 对话必须推进情节或揭示性格
- **钩子廉价化**: 每章结尾都是"突然门被推开"→ 钩子类型需轮替
- **场景过多**: 每章切换 6+ 场景，拍摄成本爆炸 → 场景精简，每场景有明确的戏剧功能
- **内心独白过载**: 用内心独白解释情感 → 情感变化通过可拍摄动作表达，每次内心独白≤3 句

## 与其他 skill 的交互

- 本 skill 定义对话驱动和视觉化写作的叙事风格约束。可与任何流派 skill 叠加
- 与 ensemble 叠加时: 每场景 ≤4 人的独立约束仍然生效，每场景人数仍受此约束
- 场景追踪.md 在本 skill 中尤为重要——场景复用率高时，每次描写的视觉一致性不能出矛盾
