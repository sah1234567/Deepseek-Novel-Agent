# 轻小说异世界写作规范

## 目标

确保日式异世界/轻小说的世界观内部自洽——RPG 游戏规则作为世界自然法则而非外部系统、
冒险者公会作为社会机构有其运作逻辑、外挂能力有明确机制与限制而非万能、
轻小说叙事风格与角色塑造保持日式特色。

**与中国系统流的关键区别:**
- 中国系统流：系统是绑定的外部工具，只有主角/少数人有，世界本身不按游戏规则运行
- 日式异世界：RPG 规则内化于世界——状态面板/等级/技能是世界的自然法则，人人都有（或至少冒险者都有）
- 中国系统流侧重：任务→奖励→成长
- 日式异世界侧重：在这个"游戏化的世界"中生活与冒险——日常与战斗并重

## 建议创建的知识库文件

单独使用本 skill 时建议: 战力系统.md（等级体系/技能树/职业系统）、势力追踪.md（冒险者公会/王国/魔王军）。

## RPG 世界规则约束

- 等级/状态/技能体系在战力系统.md 中明确定义: 等级上限/每级所需经验/属性项的含义/技能习得条件
- 状态面板（Status）的可见性需前期设定: 所有人可见/只有持有者可见/只有特定道具后可见/只有主角可见。设定后不可更改
- 经验值获取逻辑需自洽: 击败敌人≠经验值——谁造成最后一击? 组队如何分配? 非战斗行为（生产/交易）能否获得经验? 设定后在叙事中保持一致
- 技能习得方式需多样化: 等级提升自动习得（基础技能）/技能书使用（稀有技能）/师匠传授（特殊技能）/自力开发（原创技能）。不能全部是"等级到了自动获得"
- 状态异常（毒/麻痹/混乱/即死等）的机制和解除方式需设定后保持前后一致

> 例：主角10级→升11级需要100经验→击败5只史莱姆(每只20经验)+组队分配:主角70%队员30%→主角获得70经验→仍需30经验→找不到怪物→去公会接采集任务→非战斗任务也能获得少量经验(10经验)→还差20经验→发现隐藏洞穴的稀有蘑菇(采集+20经验+稀有素材)→升级→获得新技能「风刃」

## 冒险者公会约束

- 公会组织架构需合理: 公会总部→城市支部→乡镇办事处。每级的功能/权限/人员配置不同
- 冒险者等级（Rank）的晋升条件: 任务完成数/成功率/特殊考核/实力认定。同级内分上中下位以体现成长
- 任务板（Quest Board）的任务等级与冒险者等级对应: F级只能接F级任务，除非有高Rank冒险者带队。越级接任务需特殊批准
- 任务的报酬需与难度/危险度/距离/稀有度匹配: 同Rank任务之间的报酬需有合理差异——采集任务<讨伐任务<护卫任务<探索未踏区域
- 公会的政治立场: 公会名义中立，但受王国/贵族/教会的压力。公会会长的政治角力是可能的暗线

> 例：F级(新人)→完成10件F级任务+成功率≥80%→晋升E级→E级可接讨伐低级魔物任务→D级需通过公会考试(实战+笔试)→C级可任队长带低级冒险者→B级可接跨城市任务→A级可入未踏区域→S级为传奇级(全国≤10人)。主角有外挂可以加速但不能跳过——毕竟公会要看实绩

## 外挂/作弊能力约束

- 主角的外挂能力（Cheat Skill）必须在首次出场时完整声明: 能力名称/效果/触发条件/使用限制/冷却时间/代价
- 外挂不能无限进化: 即使能力可以升级，升级路径也需在首次出场时暗示或设定。不能"每当主角陷入困境就恰好解锁新功能"
- 外挂的"代价"必须存在: 即使是看起来无代价的能力，也有隐性代价——社会代价（被怀疑/被觊觎/被利用）/情感代价（孤独感/无法与普通人共情）/能力本身的不完整性（只能用于特定条件）
- 主角的独特性和"外挂"需要区分: 外挂是主角的额外优势——但主角自身的判断力/成长/人际关系才是解决核心问题的关键。不能让"因为我有外挂所以一切解决"

> 例：主角转生获得「无限成长」外挂→效果:每击败一个敌人永久增加微量全属性→限制:增加量=敌人等级的0.1%，击败低级敌人几乎不增长→代价:属性增长过快导致身体难以精确控制(第5章捏碎杯子/第10章误伤队友)→需要精神修炼来匹配肉体强度→外挂提供了成长速度，但"控制力"需要主角自己磨练

## 轻小说叙事风格

- 叙述口吻可以比传统小说更口语化/轻松: 允许主角的内心吐槽（ツッコミ）/自嘲/对读者式的解释。但同一章内口语化段落和严肃段落之间的切换需有过渡——不能前一秒在搞笑后一秒在悲壮
- 第一人称或近似第一人称的第三人称限制视角是轻小说的主流: 读者通过主角的"滤镜"看世界——主角不知道的事情读者也不知道。不能突然切换到配角视角（除非作为特殊章节的间章）
- 日常回和战斗回的交替: 每 2-3 章的战斗/紧张情节后需插入≥1 章的日常/生活描写（做饭/购物/与同伴闲聊/探索城镇）。日常回不是"水字数"——是角色塑造和世界观展示的重要场景
- "空气感"与"氛围": 轻小说的魅力之一在于场景的氛围描写——傍晚的公会大厅/雨天的书店/热气腾腾的浴场/深夜的露台对话。不是纯审美描写——氛围承载叙事功能

## 角色塑造的日式特征

- 角色的"属性化"是轻小说的特色——允许用标签化属性快速建立角色印象（傲娇/天然/无口/热血笨蛋/腹黑等），但需要在后续章节中打破标签——展示角色的多面性
- 主角的"平凡性"与"特别性"并存: 轻小说主角通常在日常生活中平凡甚至废柴，但在关键时刻展现特别的一面。这个落差需要铺垫而非突然
- 配角不是主角的陪衬: 每个队友/同伴需要有独立的目标和成长线。配角的烦恼和困境需要独立于主角——不能所有配角的人生意义都与主角绑定
- 反派需要有"可以理解的动机": 轻小说的反派很少纯粹邪恶——魔王可能有其痛苦/叛徒可能被胁迫/对手可能有自己的正义。这是与中国爽文"反派就是用来打脸"的重要区别

> 例：傲娇女骑士→初期:嘴硬+对主角不屑→中期展示:她的"傲"源于家族荣誉的压力，"娇"的一面只在极为信任的人面前出现→后期:学会坦率表达关心但仍保留好胜心→她的成长弧是"学会依赖他人"而非"被主角征服后变温柔"

## 常见错误

以下错误会导致轻小说异世界失去特色或沦为低配中国穿越文，必须避免：

- **RPG规则装饰化**: 状态面板只在前3章出现，之后就忘了 → 等级/技能/状态的变动需持续追踪，至少每10章更新一次
- **冒险者公会万能化**: 公会同时是银行/邮局/法院/政府 → 公会只需承担任务中介+等级认证+冒险者福利三大职能，其他功能由别的机构承担
- **外挂无限膨胀**: 外挂从"略强"膨胀到"无解" → 外挂能力首次出场时完整声明+升级路径预留但不能无限、需要≥1次的代价/限制展示
- **日常与战斗割裂**: 日常=水字数的无聊章节 → 日常回是角色塑造和世界观展示，每2-3章战斗后插入≥1章日常
- **角色标签化石化**: 傲娇永远只会"哼"、无口永远只会"……" → 标签是角色的入口而非全部——后续章节需展示角色的多面性和成长
- **异世界=欧洲中世纪**: 所有异世界都是城堡+骑士+魔法的欧洲中世纪模板 → 异世界可以有不同的文化基础（和风异世界/沙漠异世界/海洋异世界/魔法工业革命时代等）

## 与其他 skill 的交互

- 本 skill 与中国 system skill 兼容但不重叠: system skill 定义外部系统界面的任务/积分/奖励规则，本 skill 定义世界内化的 RPG 规则和公会制度。两者叠加时，主角的外挂(本 skill)和系统(system skill)是两套不同的机制——需明确各自的触发方式和边界
- 与 transmigration 叠加时: transmigration 提供穿越者的身份适应和知识应用，本 skill 提供异世界的 RPG 规则——同一个人被以不同方式送入异世界（转生/召唤/转移），叠加后转生方式约束由 transmigration skill 处理
- 与 romance 叠加时: 轻小说中的后宫/恋爱关系有其特有的日式节奏——"迟钝系主角"是常见设定，但迟钝需有合理原因（并非真正迟钝而是有意无视/害怕打破关系平衡）
- 与 power-fantasy 叠加时: 异世界的外挂与爽文节奏兼容——但外挂的代价/限制不可被爽感覆盖，RPG 世界的规则约束不可被"爽"打破
- 本 skill 的轻小说叙事风格（口语化/日常回/空气感）与 male-oriented/female-oriented 叠加时: 叙事风格约束优先于读者向定位——轻小说的叙事节奏是跨性别的
