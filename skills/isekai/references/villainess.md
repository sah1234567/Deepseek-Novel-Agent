# 恶役令嬢

转生为乙女游戏/小说中的反派千金——需要在命运（破灭END）到来前改变自己的结局。核心：命运回避策略、原主角与攻略对象的互动规则、恶役身份的社交困境、"游戏知识"的时效与蝴蝶效应。

## 命运回避策略

- 恶役令嬢的"破灭END"在故事开始时需明确（死亡/流放/没落/强制结婚等），回避的目标越具体越好
- 回避策略的梯级: 第一级:改变自己的行为（不再欺负女主角/不再参与恶役行为）→第二级:改变关键事件的结果（阻止某个会导致破灭的事件发生）→第三级:改变自己与关键人物的关系（让攻略对象/权力者不再敌视自己）→第四级:改变游戏本身的底层逻辑（世界不是游戏？/游戏的规则可以被打破？）
- 每个阶段的策略转换需有触发事件——不能"我突然决定改变一切"
- 回避过程不是一条直线: 越回避→越发现命运不是单一事件而是"结构性矛盾"。原主的身世/家族/社会地位本身就是"破灭FLAG"的来源——光靠做好事不能解决
- 回避失败的章节是必要的: 展示即使努力也可能失败——让读者感受到命运的压迫感。但失败后需有新的突破口

> 例：穿越成恶役令嬢→破灭END:在学园毕业典礼上被王子公开断罪→流放→家族没落→死于边境。回避策略第一级:对女主角友善+停止欺凌→原女主很困惑但接受→第二级:阻止"陷害女主偷首饰"事件→但命运用了另一种方式（有人冒充恶役令嬢的名义陷害）→第三级:直接向王子坦白"我过去做错了什么+我为什么要改变"→王子半信半疑→第四级:发现这个世界不只是游戏——有游戏剧本以外的隐藏势力在推动事件

## 原主角与攻略对象

- 原女主角≠白莲花: 原女主有独立人格——可能是善良的/可能是腹黑的/可能是被命运推着走的。不能因为是"女主角"就默认完美无缺
- 攻略对象的性格和好感度逻辑需尊重原作设定: 每个攻略对象在原游戏中的攻略条件和好感事件——这些信息决定恶役令嬢的"攻略知识"。但攻略知识会过期（见下）
- 攻略对象对恶役令嬢的态度不是"全员厌恶": 不同攻略对象的初始态度应有差异——有人公开敌视/有人表面礼貌但疏远/有人对传言持保留态度/有人暗中同情。初始好感度的差异决定了关系的起点
- 恶役令嬢"攻略"的不是恋爱而是生存: 目标是"不被断罪"而非"和王子谈恋爱"。即使陷入恋爱，那也是结果而非目标——混淆这点的恶役令嬢故事会失去紧张感

## 社交困境

- 恶役令嬢的"坏名声"在故事开始时已经成为社会共识——不是"改变行为=改变评价"。名誉的修复需要≥10章的长期经营和≥3个关键事件
- 社交场合中的信息不对等: 恶役令嬢知道"接下来会发生什么"（游戏剧情知识）但无法解释为什么知道——因此她的行动在旁人看来可能"突然改变/不合常理"
- 家族压力的约束: 恶役令嬢通常是贵族之女——家族有自己的利益/政治立场/对女儿的期望。不能无视家族只做自己想做的事
- 学园/社交季等封闭舞台的约束: 角色被限定在特定空间和社交圈内，自由度受限——这既是情节的推动力也是限制

## 游戏知识的时效

- 游戏剧情知识的准确性随时间衰减: 第一年（玩家已知的学年）剧情高度准确→第二年起蝴蝶效应开始显现→第三年"剧本"已经面目全非
- 主角知道的信息仅限于"玩家在游戏原作中能获取的信息": 不知道角色的隐藏背景（游戏续作/DLC/设定集的内容）、不知道游戏剧本之外的秘密、不知道"通关后"这个世界的走向
- 隐藏攻略对象/里路线/续作信息: 如果主角玩的只是原作，那么"续作/加强版/动画原创"的信息她不知道。这是天然的情节转折来源——"我以为我知道一切但我只知道一半"
- 游戏脚本≠现实: 游戏中被省略的日常（角色在非事件日做什么/角色的家庭关系/政治背景/经济状况）——这些是游戏没展示但现实中存在的维度

## 常见错误

- **回避过易**: 做了两件好事→所有人立刻原谅+爱慕 → 名誉修复需≥10章的经营+≥3个关键事件，社交标签不是那么容易翻的
- **原女主工具化**: 原女主要么=完美白莲花要么=隐藏恶女 → 原女主需有独立人格和在恶役令嬢"介入"后的独立反应
- **攻略对象全员倒贴**: 所有攻略对象都爱上恶役令嬢 → 关系方向需多样化(同盟/亦敌亦友/师友/同行者)，不能全部是恋爱关系
- **游戏知识永不到期**: 到故事后期还能准确预测事件 → 蝴蝶效应随时间加剧，第二年剧情开始偏离，第三年剧本面目全非
- **恶役设定即甩锅**: 所有的恶役行为都是"原主做的"→主角完全没有原主的阴影 → 原主的行为后果是主角必须承担的，不能一键清零。至少1次因原主过去的行为而遭遇实质代价
