# 娱乐圈写作规范

## 目标

确保娱乐圈作品逻辑自洽——公众形象变化有事件驱动、粉丝反应符合阶段特征、
舆论危机有合理的引爆和化解过程、行业等级梯度分明、同行反应多样化。

## 建议创建的知识库文件

单独使用本 skill 时建议: 时间线.md（关键事件/作品发表/舆论节点）。
人物卡必须启用: 公众形象演变日志（形象阶段/粉丝规模/舆论倾向）。
视剧情: 势力追踪.md（经纪公司/工作室/投资方）。

## 职业等级与行业梯度约束

- 主角的职业发展必须有清晰的等级梯度，不能从龙套一跳成为影帝。每个等级标注: 作品积累门槛/行业认可度/收入水平/粉丝规模
- 等级跃升需要触发事件: 代表作爆发/获奖/重量级合作/舆论逆转——不能无事件跳级
- 行业内的"咖位"体现在实际待遇中: 片酬/番位/代言等级/综艺镜头量/红毯顺序
- 不同细分领域（影视/音乐/综艺/时尚）的等级不完全互通——电影大咖在音乐领域可能是新人

> 例：演员等级梯度 → 群演(Ch.1-3)→特约(Ch.4-8,首个小角色)→配角(Ch.9-15,首次有台词角色)→重要配角(Ch.16-22,被观众记住)→主角(Ch.23+,首部主演)→一线(代表作+奖项)→顶级(票房/奖项/国民度三项齐全)

## 公众形象管理约束

- 公众形象演变日志记录: 当前形象标签/粉丝规模/路人好感度/黑粉活跃度/媒体关系
- 形象变化必须由具体事件驱动: 人设崩塌需要丑闻或失误，形象翻红需要反转事件
- 公众形象的演变有惯性——不能今天全网黑明天全网夸，中间必须有过渡
- 媒体关系是独立维度: 有无固定合作的媒体/有无对家媒体/与狗仔的关系

## 粉丝与舆论约束

- 粉丝规模和构成随主角发展阶段变化: 初期靠作品吸粉→中期靠人设固粉→后期靠国民度
- 粉丝行为可以影响主角的公众形象——粉丝互撕/应援/控评的反噬
- 舆论危机的引爆需要合理源头: 对家抹黑/旧事被挖/言行失误/作品翻车
- 舆论危机的化解必须有实质行动: 道歉/作品证明/时间淡化/反转事件——不能"过几天大家就忘了"
- 同行反应需多样性: 暗自幸灾乐祸/公开站队/私下安慰/借机炒作

> 例：舆论危机链 → Ch.25 主角被曝"片场耍大牌"(源头:对家工作室放料+群演匿名爆料)→Ch.26 舆论发酵(粉丝护主vs路人吃瓜vs对家粉丝推波助澜)→Ch.27 主角回应(公开道歉+放出片场花絮澄清关键细节+合作演员声援)→Ch.28-29 舆论逐步平息但路人好感度从75↓至58

## 行业运作约束

- 作品发表→业内反响→社会影响 三层递进，不能跳过中间层直接抵达社会影响
- 业内规则（番位争夺/资源置换/对赌协议/压番抬番）需在首次涉及时设定，后续遵守
- 同一个经纪公司的艺人之间存在资源竞争——不是所有人都对主角友善
- 奖项/榜单/收视率的竞争需有合理性——不能所有奖都被主角包揽

## 常见错误

以下错误会导致娱乐圈文脱离行业真实感，必须避免：

- **咖位跳跃**: 主角从群演直接到主角 → 等级梯度需分明，每次跃升需有触发事件
- **全网统一态度**: 所有人对主角要么都夸要么都骂 → 同行/粉丝/路人/媒体的反应需各有立场
- **舆论无惯性**: 今天全网黑明天全网夸 → 形象变化有惯性，中间需有过渡事件
- **粉丝零反噬**: 粉丝永远是主角的后盾而不会惹事 → 粉丝行为至少一次对主角产生负面影响
- **资源无限堆叠**: 主角同时拿下所有顶级资源 → 行业资源是有限的，得到资源=抢走别人的
