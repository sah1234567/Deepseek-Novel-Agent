# 官场写作规范

## 目标

确保官场/仕途世界观内部自洽——权力体系有明确层级与晋升规则、派系博弈
有完整利益链和因果、信息战传播路径可追溯、施政决策有反馈周期、
不出现权力凭空获得或政治规则突变等矛盾。

## 建议创建的知识库文件

单独使用本 skill 时建议: 势力追踪.md（派系/山头/利益集团）、时间线.md（晋升时间节点）。
_关系与称呼索引.md（官场称呼随职位变动）。
视剧情: 场景追踪.md（不同衙门的场景一致性）、因果链.md（政治事件的连锁反应）。

## 权力体系与晋升

- 职位层级在势力追踪.md 中明确定义: 每级标注品级/管辖范围/权力边界/晋升硬性条件（年限/政绩/考试）
- 晋升链条必须完整: 政绩积累→关键事件（机会窗口）→上级赏识（派系支持）→公示/任命→权力交接。缺任何一环都视为晋升铺垫不完整
- 破格提拔需≥10 章铺垫+重大政绩事件支撑——不能"因为主角厉害所以连升三级"
- 降职/罢免需有合理触发: 政敌攻击/站错队/重大事故追责/替上级背锅。降职后恢复职位需比首次获得更艰难
- 职位变动后在≥3 章内展示实际影响: 权力范围变化/下属态度转变/派系重新拉拢或疏远

> 例：李知府从正五品升从四品→政绩:辖内税粮增收三成(Ch12-15)→关键事件:巡抚巡视时应对得当(Ch16)→上级赏识:巡抚递话给吏部(Ch17)→任命下达(Ch19)→权力交接:移交原职+赴任新地+旧部惜别/新下属试探(Ch20-22)

## 派系博弈

- 派系在势力追踪.md 中记录: 核心人物/利益诉求/敌对派系/当前阶段（上升/稳定/衰落）
- 每个政治事件需分析三方利益: 谁获益/谁受损/谁中立观望
- 派系支持不是永久的: 利益一致时结盟→利益冲突时分裂→每≥15 章至少一次派系格局变动
- 主角的政治立场需明确: 依附某个派系/独立自成一派/在各派之间周旋。立场切换需≥5 章铺垫+重大触发事件
- 博弈手段需合法合规包装: 即使是不正当竞争，也必须通过合法程序/规章制度的外壳来实现

## 信息战与舆论

- 官场中的信息是关键武器: 谁掌握什么信息记录在人物卡已知信息演变日志
- 消息传播链需可追溯: 源头→传递者→接收者→决策者。不能"朝中都在传"而不交代谁传的
- 消息的真实性和时效性需区分: 有人故意放虚假消息（烟雾弹）/有人压着真消息不放（信息钳制）/有人提前获取消息（信息优势）
- 舆论风向的转变需有触发事件+≥3 章发酵过程——不能今天舆论支持明天舆论反对
- 告密/举报/弹劾需有完整动因: 利益驱动/派系授意/自保/正义感。匿名举报的幕后推手需在≥5 章后揭示

## 施政与反馈

- 每项重大政策的提出→讨论→施行→反馈需≥5 章跨度——不能今天下令明天见效
- 政策效果有延迟和意外: 好政策可能产生坏副作用，坏政策可能有短期红利
- 施政阻力来源需具体: 既得利益者反对/下级阳奉阴违/资源不足/政策本身缺陷——不能"大家都支持就是执行不下去"
- 政绩的归因需合逻辑: 一项政策的成功/失败不能全部归因于主角——但主角的关键决策必须起核心作用

## 常见错误

以下错误会导致官场沦为儿戏或脱离现实，必须避免：

- **晋升坐火箭**: 主角3章一升、5章一跨级 → 每次晋升需完整链条（政绩+机会+赏识+任命），破格提拔需≥10章铺垫
- **对手降智**: 政敌都是蠢货，主角随便设局就中计 → 对手必须有自保退路和反击手段，政治斗争是双向博弈
- **政策魔法**: 主角一拍脑袋→政策完美执行→全民叫好 → 政策有反馈周期（≥5章），有好有坏的副作用
- **派系脸谱化**: 主角所在派系=好人，对立派系=坏人 → 每个派系有合理利益诉求，正邪不绝对
- **消息全知**: 主角总能第一时间获取所有关键消息 → 信息获取需有合理来源（眼线/情报网/人际关系），信息优势和劣势交替

## 与其他 skill 的交互

- 本 skill 定义的权力体系独立于修炼/战力体系——当与 setting 类 skill 叠加时，权力体系和社会地位是平行的第二重进阶线
- 与 palace（宫斗/宅斗）兼容但不重叠: palace 关注后宫/内宅的微观权力，political 关注朝堂/体制的宏观权力。两者叠加时，后宫斗争可作为朝堂博弈的延伸
- 与 gangster（黑道）互斥——不可叠加。官场以体制内权力为中心，黑道以地下秩序为中心。如混合元素（黑白勾结），选择占比≥70%的一方作为主 skill
- 当与 romance 叠加时: 官场中的婚姻有政治联姻属性——情感关系受权力格局约束
- 势力追踪.md 在本 skill 中记录派系/山头/利益集团
