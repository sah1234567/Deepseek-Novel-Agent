# 国运流写作规范

## 目标

确保国运流体系自洽——国运奖惩规则前期声明后不可更改、奖罚幅度与国家体量匹配、
代表选拔和更替有明确机制、全球信息传播逻辑一致、国内民众反应和国际博弈真实可信。

## 国运流的核心特征

以下特征区分国运流与无限流/竞技文/末世文：

- **国运绑定**: 代表的胜负直接影响国家命运。胜利=国家获得资源/领土/祝福/技术；失败=国家遭受灾害/资源枯竭/领土缩减/人口削减。奖惩必须是国家级的，而非个人得失
- **全球直播**: 挑战过程对全球公开。全世界的民众、政府、媒体都能观看——这是爽感的核心来源：被低估→逆袭→全球震惊
- **代表制**: 每个国家派出代表参赛。代表可以是天选（神秘机制选定）、自愿报名、或国家推举。代表的更换机制需前期设定
- **文明级奖惩**: 奖励和惩罚的尺度是文明级的——"某国淡水总量增加30%""某沿海城市被海啸吞没""某国领土扩张15%"，不能只是"获得一件道具"
- **国际博弈**: 各国代表的胜负构成新的国际格局。强国想维持优势、弱国想逆袭翻盘、邻国之间的牵制和联盟——国际关系随每一轮挑战结果动态变化

## 建议创建的知识库文件

单独使用本 skill 时建议:
- 国运规则.md — 游戏规则（参与机制/挑战周期/奖惩体系/各国代表名单/国家排名），这是本 skill 最核心的文件
- 势力追踪.md — 每个国家作为一个势力，追踪：代表/国运积分/已获奖励/已受惩罚/排名/国际联盟关系
- 场景追踪.md — 每次挑战的场景和环境描写
- 时间线.md — 挑战的时间节点、国运奖励的落地时间

视剧情:
- 道具追踪.md — 有国运道具/禁术等设定时
- 人物卡: 各主要国家的代表作为独立人物卡

## 国运奖惩体系约束

国运奖惩是本流派最核心的规则，必须在首次出场时完整声明：

**奖惩规则声明**:
- 奖惩触发条件: 什么算胜利？什么算失败？排位制还是通过制？平局如何处理？
- 奖惩幅度范围: 最小奖惩和最大奖惩的边界——不能第一场奖励"淡水+5%"第五场奖励"获得永恒能源"
- 奖惩落地机制: 资源如何增加？（凭空出现/地质变化/气候改变）领土如何扩张？（陆地升起/海域退去/国界平移）需要设定至少一个"落地逻辑"，且之后始终一致

> 例：国运规则声明 → 每轮挑战排名前30%的国家获得"国运赐福"，后30%受到"国运惩罚"，中间40%不变。奖励幅度与排名成正比，惩罚幅度与排名成反比。资源奖励通过"地质缓慢变化"实现（如矿脉在三个月内逐渐生成），领土变化通过"海平面升降"实现。

**奖惩与国家体量的平衡**:
- 小国获得"淡水+50%"可能只是几条河，大国获得"淡水+5%"可能是一条大江——奖励的实际绝对值需与国家体量匹配
- 同样的挑战排名，不同体量国家的奖惩幅度应不同——否则小国永远无法追上大国，故事失去悬念
- 奖惩不能只给好处不给代价：获得新领土→边境摩擦增加；获得资源→邻国觊觎加剧

> 例：小国A（人口500万）获得"粮食产量+40%"→ 全国粮食从勉强够吃到有出口盈余；大国B（人口3亿）获得"粮食产量+10%"→ 绝对值大于A国但比例较小。小国用比例追赶，大国用体量维持，形成动态平衡。

## 代表选拔与更替约束

- 代表选拔机制需在开局明确: 随机天选/国家推举/自愿报名/前世记忆觉醒者？标准是什么？
- 代表能否主动退出？退出后是否有惩罚？
- 代表死亡/失去资格后如何替补？替补是否有适应期或能力惩罚？
- 代表在挑战之间的状态: 可以回到现实生活？还是被隔离？
- 国家政府与代表的关系: 政府可以指挥代表吗？代表是否有独立决策权？

> 例：选拔规则 → 每国随机天选3人，其中1人被指定为"主代表"。主代表死亡或主动放弃后，由剩余2人中积分较高者递补。下一轮挑战前若3人均无法参赛，该国本轮直接判负。代表在挑战间歇可返回正常生活，但下一轮挑战开始前24h必须进入备战状态。

## 全球信息传播约束

全球直播是国运流爽感的核心机制，需要前后一致的传播逻辑：

- **传播方式**: 全球天空出现巨幕？还是每个国家有一个"国运石碑"？还是通过手机/电视直播？
- **信息粒度**: 全球观众能看到什么？挑战全过程（包括代表私下对话）？还是只有关键节点？不同透明度导致不同的故事可能
- **观众反应机制**: 国内民众能否以某种方式支持代表（信仰之力/弹幕加油/祭祀）？如果可以，机制需固定
- **国际舆论**: 各国媒体/政府/网民的反应应有多样性——崇拜/恐惧/嫉妒/学习/阴谋论

> 例：传播设定 → 全球天空同步出现巨型光幕，24h不间断播放所有挑战。观众只能看，无法与代表互动。每个国家的光幕默认显示本国代表视角，但可切换至他国视角——这导致各国观众对本国的关注度天然更高，国际舆论通过社交网络和新闻发布会等形式出现。

## 国际博弈约束

- 各国排名变化记录在势力追踪.md，排名变化必须触发国际关系变化——不能排名翻天覆地而外交格局纹丝不动
- 强国衰落后的反应: 不甘/反扑/寻找漏洞/联合其他衰落国；弱国崛起后的挑战: 被围堵/被拉拢/被要求承担责任
- 代表之间的私人关系与国家关系可以不同——A国代表和B国代表是朋友，但两国处于敌对排名竞争——这种张力是剧情燃料
- 结盟/背叛/交易必须是实质性的: 结盟=共享部分情报或资源但有代价，背叛=打破规则有后果，不能只是"嘴上说联盟"

## 常见错误

以下错误会导致国运流沦为儿戏，必须避免：

- **奖惩规则飘移**: 前期胜利奖励是"粮食增产"，后期同样级别的胜利突然奖励"反重力科技"→ 奖惩幅度和类型需在国运规则.md前期声明，大幅偏离需有"游戏升级/阶段变更"等大事件支撑
- **奖惩无落地逻辑**: 宣布"获得新领土"但不说明领土从哪来，邻国毫无反应 → 每次领土级奖惩需说明落地方式和对周边的影响
- **代表选拔随意化**: 前期说是随机天选，后期主角能用关系把自己人选进去 → 选拔规则一旦设定不可被角色钻空子，除非钻空子本身就是有代价的剧情
- **全球反应单一化**: 所有国家要么崇拜主角国要么敌视主角国 → 每个主要国家应有独立的利益考量和态度演变，态度变化需要事件驱动
- **小国无限逆袭**: 小国每轮都拿第一，10轮后资源总量超过所有大国之和 → 奖惩幅度与体量匹配，小国靠比例追赶但不能突破资源总量的物理逻辑
- **挑战间歇无事发生**: 挑战间歇只用来日常闲聊 → 挑战间歇应推进：国际关系演变/国内政局变化/上一轮奖惩的落地效果/下一轮情报收集

## 与其他 skill 的交互

- 本 skill 定义国运奖惩、代表选拔和全球信息传播的基础规则。与 infinite 的边界：无限流的副本奖惩是**个人积分**（用于兑换个人能力），国运流的奖惩是**国家命运**（领土/资源/人口），两者不互斥可叠加——无限流参与者恰好被天选为国运代表
- 本 skill 天然具有爽文属性，与 power-fantasy 叠加时：打脸节奏、反派梯度、爽点密度等约束自动生效。但国运奖惩规则不可被爽文规则覆盖——不能为了爽让国家凭空获得远超设定范围的能力
- 与 farming 叠加时：国运奖励可能包含"获得未开发领土""获得新资源"——后续建设由 farming skill 约束
- 与 ensemble 叠加时：若采用多代表制，不同代表的视角和戏份按群像规则平衡
- 势力追踪.md在本 skill 中承担"国家排名/联盟关系/已获奖惩"的追踪——不同于其他 skill 的势力（门派/家族/公会），是国家级的势力管理
