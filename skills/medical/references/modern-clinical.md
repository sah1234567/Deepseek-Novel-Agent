# 现代临床

现代医院/西医/外科/急诊等现代医学题材。核心：手术室的真实氛围、医院内部的职场生态、医患关系的结构性张力、医学伦理的灰色地带。

## 手术室场景

- 手术室的氛围描写需有具体感官细节: 消毒水的气味/监护仪的滴答声/无影灯的冷光/器械碰撞的金属声/止血纱布的温度。每次手术描写至少涉及3种感官
- 手术团队的配合: 主刀→一助→二助→器械护士→巡回护士→麻醉医生。每个人有自己的职责和站位。手术中的沟通以主刀为中心——主刀的命令需精确到器械名称和动作
- 手术中突发状况的应对流程: 异常发现（如发现肿瘤比术前评估更严重）→主刀暂停→与麻醉医生和助手商议→决定改变术式或继续→术中请会诊（如需）。决策链的每一步都需有合理的时间压力和医学逻辑
- 手术后的环节不可省略: 关腹/缝合→病人转PACU→术后医嘱→与家属沟通手术结果→手术记录→术后查房。至少展示2个环节

> 例：肝切除手术→开腹:电刀切开皮下→暴露:游离肝周韧带→关键操作:阻断肝门(Pringle maneuver，每次≤15分钟)→切除:沿预切线离断肝实质→止血:氩气刀+止血纱+缝合→关腹:检查无活动性出血→逐层关腹→术后:ICU监护24小时→第3天排气→第5天拔管→第7天出院

## 科室差异与职场生态

- 不同科室的医生有截然不同的性格画像和工作节奏: 急诊科（快节奏/高压/什么都见过/嘴毒但靠谱）/外科（自信到自负/重视技术/鄙视内科的"只会开药"）/内科（思考型/重视诊断过程/认为外科"只会切"）/ICU（悲观主义者/永远做最坏打算）/儿科（超强耐心+沟通能力/承受来自家长的压力）
- 科室间的会诊是天然的跨科室互动: 外科请内科会诊（病人有内科合并症）→内科医生来外科病房→两个科室的思维碰撞→既可能合作也可能摩擦
- 医院行政与临床的张力: 医务科/院办关心的是"不出事/指标达标/经济效益"，临床医生关心的是"治好病人"。两者的目标有时一致有时冲突——这是医院政治的核心
- 规培生/住院医的"底层"体验: 写不完的病历/被上级骂/值不完的夜班/被护士怼（护士比住院医更有经验是常态）/微薄的工资/不知道这样的日子还要多久。这是医学文最能让医学生/年轻医生共鸣的部分

## 医患沟通

- 坏消息告知的场景极其重要: 告知癌症诊断/告知手术失败/告知死亡。SPIKES六步法可作为框架: 设定(Setting)→探知患者认知(Perception)→邀请告知(Invitation)→传递信息(Knowledge)→共情回应(Empathy)→策略与总结(Strategy)
- 患者的"非医学语言"与医生的"医学术语"之间的翻译: 好医生能用患者理解的话解释病情——"你的心脏的泵血能力只有正常人的40%，就像一辆车只剩40%的动力，平时能开但上坡很吃力。"——这是医学文的黄金描写
- 家属≠患者: 家属的决定有时与患者利益冲突（如家属要求隐瞒病情、要求不惜一切代价治疗而患者已无力承受）。医生被夹在中间——这是天然的伦理困境

## 常见错误

- **手术描写过于血腥***: 写成恐怖小说 → 手术描写控制在专业技术范畴，不渲染血腥和疼痛感。重点在"决策+操作+团队配合"而非"割开有多痛"
- **所有手术都成功**: 主角=神话 → 手术失败/并发症/术后感染是医学现实，每10章至少1次不完美结果
- **医患关系二元化**: 医生=好人/病人=无理取闹 → 最有张力的医患冲突来自结构性问题（信息不对称/经济限制/制度缺陷），而非个人善恶
