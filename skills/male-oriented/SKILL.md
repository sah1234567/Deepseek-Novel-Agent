# 男频写作规范

## 目标

确保男频小说的叙事节奏和角色塑造满足男性读者的阅读期待——
事业成长线作为主轴、爽感节奏有起伏、女性角色塑造有辨识度、
兄弟情义和竞争叙事推动主线。本 skill 是与题材 skill 叠加的**读者向元规则**。

## 建议创建的知识库文件

与具体题材 skill 叠加使用，无额外文件需求。
男性角色的成长记录在人物卡修为/能力演变日志中。

## 核心原则: 男频的叙事重心

男频小说的三大叙事支柱:

1. **事业成长线优先**: 主角的能力/地位/势力持续上升是核心驱动力。情感线是副线，不能拖慢事业线节奏
2. **爽感达成节奏**: 每≤3 章至少一次"爽点"——能力突破/打脸对手/获得资源/地位晋升/智谋碾压
3. **竞争即推动力**: 主角通过竞争（战斗/博弈/排名）证明自己，竞争者=情节推进器

**与女频的关键区别:**
- 男频: 外部世界征服→内部情感反应。先写"他做了什么"，再写"他怎么想"
- 女频: 内部情感体验→外部世界反应。先写"她感受到什么"，再写"她怎么做"
- 男频的感情线是**成果展示**（获得优秀伴侣=实力的证明）
- 女频的感情线是**过程体验**（情感的起承转合=阅读快感来源）

## 事业成长线约束

- 主角的能力/地位必须呈阶梯式上升——每卷结束时处于比开卷时更高的位置
- 成长必须有可见的量化: 等级提升/排名上升/势力扩张/收入增长/声望提高——至少追踪 2 个维度
- 瓶颈期≤10 章: 主角不能长时间停滞，即使遇到瓶颈也需在 10 章内找到突破方向
- 每次突破后的"展现实力"环节不可省略: 新能力/新地位必须在获得后≥1 章内展示实际效果
- 成长的代价需有但不可喧宾夺主: 可以受伤/付出/被误解，但代价描写篇幅≤成长描写篇幅的 40%

## 女性角色塑造

**核心定位:** 男频中女性角色为男主叙事服务是**类型常规而非缺陷**。女性角色作为男主实力的证明/征服的奖励/团队的资源——这本身就是男频读者的核心爽点来源。关键不在于"工具化是否合理"（合理），而在于**工具是否好用**——即角色是否有辨识度、出场退场是否前後一致、她们对男主的贡献是否可感知。

- 每个女性角色首次出场时需有≥1 个**独特标识**: 特殊能力/特殊身份/口癖/外型特征/行为模式/与男主相遇的独特方式。标识越鲜明，读者记忆越深——后宫人数越多越需要差异化
- 女性角色的"类型分工"是合理的叙事工具: 战力型/治愈型/智囊型/傲娇型/温柔型/神秘型——不同类型满足不同读者偏好。但同类型角色≥2 人时必须有细分差异(如同是傲娇，一个偏向口是心非、一个偏向高冷外表下的关心)
- 征服后的"能力保持"是维持征服价值的需要: 加入前是天才剑客→加入后只会端茶倒水——这削弱了读者的征服满足感，因为奖励贬值了。保持她的能力并在男主团队中发挥作用=征服了一个"有价值的人"而非"用后即弃的道具"
- 女性角色之间的关系以"围绕男主"为主轴: 争风吃醋是合理的戏剧冲突——但若后宫人数≥4，至少引入 1 种非竞争关系（如两人联手帮男主解决某个问题），避免全部女性角色的互动模式同质化
- 后宫成员的**可见度管理**: 加入后不能彻底消失，但也不必每章都出场。每位后宫成员每 5-10 章至少有 1 次有台词的出场——让读者记得"她还在"

> 例：苏婉清→标识:冷面毒师(用毒高手)+口头禅"无聊"+与男主相遇:男主重伤闯入她的药庐→加入后:继续炼毒+为男主的战斗提供毒攻支援+因不善表达感情产生误会→她的"毒师"标签始终在，对男主的贡献可感知——这就是"好用的工具"

## 兄弟情义叙事

- 兄弟/战友关系需有建立过程: 初始对立/不信任→并肩作战→互相信任——不能第一面就"为兄弟两肋插刀"
- 兄弟角色需有分工: 智囊型/战力型/搞笑担当/后勤支持——不能全是"和主角一样能打的好兄弟"
- 兄弟的独立支线需存在: 兄弟有自己的目标/困境/成长——不能只是主角的跟班
- 兄弟牺牲/背叛是重头戏: 牺牲需≥10章铺垫，背叛需完整动机链（见 gangster skill 忠诚与背叛约束）
- 兄弟情≠基情（除非用户明确要求 BL 元素）: 描写重点在"信任"和"牺牲"而非暧昧

## 竞争叙事与对手塑造

- 每个阶段必须有明确的竞争对手——不能主角一人独孤求败
- 对手梯度: 同级竞争者→上一级压制者→跨级挑战者。每梯度≥1 人
- 对手必须有合理实力: 不能"设定很强但一碰就碎"。对手的失败需有合理解释（主角智谋/新能力/团队配合/对手自身失误）
- 对手可以转变为盟友——但转变需重大事件触发（如共同对抗更强敌人），转变后需≥3 章磨合

## 节奏与爽点密度

- 每章≥1 个微爽点: 小突破/新信息/小打脸/新资源获取
- 每≤3 章≥1 个中爽点: 战胜对手/等级突破/重要人物赏识
- 每≤10 章≥1 个大爽点: 越级胜利/重大突破/身份揭晓/关键角色收服
- 连续爽后需缓冲: 每 3 个大爽点后插入≥1 章过渡（谋划/日常/世界观展开/配角支线）
- 爽点不能连续同类型: 不能连续 3 章都是"打架打赢了"

## 常见错误

以下错误会导致男频读者流失或口碑崩塌，必须避免：

- **后宫同质化**: 后宫≥4人时所有女性角色的性格/身份/与男主的互动模式趋同 → 每个成员需≥1个独特标识，同类型角色≥2人时必须有细分差异
- **征服后贬值**: 女性角色加入后能力被遗忘/不再出场/性格模板化 → 保持其能力标签和可见度（每5-10章≥1次有台词出场），征服的价值在于"得到了一个厉害的人"而非"用后即弃"
- **成长线停滞**: 主角连续 10 章无能力/地位/势力提升 → 瓶颈期≤10章，需在10章内给出突破方向
- **兄弟全员工具人**: 所有兄弟角色=主角跟班 → 每个兄弟需有独立目标/困境/成长线
- **对手不断喂经验**: 反派一个个送，主角越来越强→对手需梯度递进(同级→压制→跨级)，每梯度≥1人

## 与其他 skill 的交互

- 本 skill 是**读者向元规则**——与任何题材 skill 叠加，不能取代题材 skill 的核心设定
- 与 power-fantasy 叠加时: 本 skill 的爽点密度约束与 power-fantasy 的打脸节奏合并——两者一致时取更激进的值，冲突时 power-fantasy 的打脸类型轮替优先
- 与 romance 叠加时: 本 skill 的女性角色塑造规则激活。后宫/种马参考文件由 romance skill 提供，本 skill 提供男性视角下的角色差异化标准
- 与 wuxia/xianxia/high-martial 等战力类 skill 叠加时: 本 skill 的竞争叙事和对手梯度约束激活——对手的梯度递进与战力体系等级对应
- 本 skill 不定义世界观/战力/情感的具体设定——只定义叙事节奏和角色塑造的读者向标准
