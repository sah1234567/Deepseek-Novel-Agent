# 灵魂互换写作规范

## 目标

确保灵魂互换规则清晰且前后一致——互换触发/解除条件、双重身份体验、
伦理边界不出现矛盾，互换双方的行为始终符合各自性格。

## 建议创建的知识库文件

必须叠加具体流派 skill。双方人物卡需启用: "身体互换日志"（互换对象/持续时间/触发/各自体验）。
关系变化记录在 _关系与称呼索引.md。

## 互换规则约束

- 互换触发条件必须在首次互换时完整说明——不能逐渐追加机制
- 触发条件保持一致: 不能 Ch3 碰法器触发，Ch10 变成情绪激动触发（除非有规则升级）
- 每次互换的持续时间需合理: 太短无意义（≤1小时），太长则涉及生理需求等实际问题
- 互换期间双方获得对方身体的记忆/习惯/技能的深度需明确设定
- 互换解除条件需设定（时间到/触发反转/第三方介入）

> 例：互换规则声明 → 触发:两人同时触碰"阴阳镜"+月圆之夜 → 持续:24h自动解除或任意一方再次触镜 → 记忆:保留自身全部记忆但无法读取对方记忆 → 技能:可使用自身技能(受对方身体限制，如魔法师魂入武士体则无法施法)

## 双重身份约束

- 双方在对方身体中的行为决策需符合各自的性格——而非互换后性格也跟着换
- 第三方对"身体中的人已变"的察觉程度需追踪（有些人可能注意到但不说）
- 互换期间的社交互动必须和常规状态的言行有可辨识的差异
- 互换对身体/精神的影响（疲劳/心理创伤/身份混淆）需在≥3次互换后展现

## 互换伦理约束

- 互换期间任何涉及对方身体的重大决定（医疗/感情/财务）需双方知情同意
- 互换期间获取的对方隐私信息→恢复后如何使用（保留/遗忘/利用）
- 如果互换涉及超过 2 人（连环互换/群体互换），规则复杂度成倍增加

## 常见错误

以下错误会导致互换失去逻辑性，必须避免：

- **互换后性格也跟着换**: 身体里是 A 的灵魂但行为变成了 B 的 → 互换是换身体不换灵魂
- **旁观者集体眼瞎**: 身体中的人明明言行大变但无人察觉 → 至少 1 个角色需要在 3 章内察觉异常
- **互换代价为零**: 频繁互换无任何副作用 → 每次互换应有身体排斥/精神疲劳/记忆混淆等代价的累积
- **互换沦为喜剧工具**: 互换仅制造性别/身份反差的笑料 → 每次互换必须推进≥1 条情节线或关系线

## 与其他 skill 的交互

- 本 skill 定义身体互换的规则框架，不定义世界观——需与至少一个世界观设定类 skill 叠加使用
- 与 transmigration 叠加时需注意边界——互换≠穿越，灵魂在另一个身体≠在不同世界
- 本 skill 规则独立于节奏控制类 skill——互换频率和时机由节奏控制类 skill 约束
