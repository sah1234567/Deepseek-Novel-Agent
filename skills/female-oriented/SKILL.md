# 女频写作规范

## 目标

确保女频小说的叙事节奏和角色塑造满足女性读者的阅读期待——
情感体验作为核心驱动力、人物关系细腻有层次、女性角色有独立成长弧光、
男性角色塑造满足期待、审美与氛围描写有质感。本 skill 是与题材 skill 叠加的**读者向元规则**。

## 建议创建的知识库文件

与具体题材 skill 叠加使用，无额外文件需求。
女性角色的情感变化记录在人物卡性格演变日志中。

## 核心原则: 女频的叙事重心

女频小说的三大叙事支柱:

1. **情感体验优先**: 人物的情感变化、关系演变是核心阅读快感来源——不是"发生了什么"而是"感受到什么"
2. **关系驱动情节**: 情节由人物关系的变化推动——冲突源于关系张力（误解/立场对立/情感纠葛/阶级差异），而非纯粹的外部事件
3. **女性角色立体成长**: 女主角的成长不仅在外在地位，更在内心成熟/情感自洽/自我认同——内在成长和外在成就并列

**与男频的关键区别:**
- 女频: 内部情感体验→外部世界反应。先写"她感受到什么"，再写"她怎么做"
- 男频: 外部世界征服→内部情感反应。先写"他做了什么"，再写"他怎么想"
- 女频的感情线是**过程体验**（情感的起承转合=阅读快感来源）
- 男频的感情线是**成果展示**（获得优秀伴侣=实力的证明）

## 情感体验描写

- 每次情感变化需有**内心反应链**: 外部事件→即时感受→身体反应→回忆/联想→行为倾向。不能跳过中间环节
- 情感描写使用"展示而非告知": 用动作/对话/生理反应/具体记忆来呈现情感，禁止抽象标签（"她很伤心"→"她攥紧了他留下的玉佩，指甲陷进掌心也没松开"）
- 情感转折需有酝酿: 从情感 A 到情感 B 不能突然——需要≥1 章的过渡（犹豫/反复/触景生情/外部推动）
- 同一章内情感温度变化≥2 次: 不能一整章保持同一情感基调——需有微妙的起伏

> 例：女主得知男主已订婚→即时感受:胸口像被攥住→身体反应:指尖发凉，端茶的手微微发抖→回忆:想起他说的"等我回来"→行为倾向:想当面质问但自尊心让她停住脚步→最终行为:默默将定情信物收进妆奁最深的抽屉

## 人物关系细腻度

- 主要人物之间的每次互动需包含≥2 层信息: 表面对话+内心活动/表面行为+真实意图。读者看到的要比角色自己意识到的多
- 关系变化需有"微小标志": 称呼的改变/距离的远近/眼神的交错/小动作/语气词变化——至少追踪 2 种微观标志
- 误解作为情节工具：需有合理信息差、持续≤5 章、消除后不可复用同一信息差
- 和解/破冰需有过程: 消除误解→尴尬期→重新试探→重建信任→关系升级。不能"误会解除=回到从前"
- 闺蜜/女性友谊线需独立于感情线: 不能闺蜜只是"情感顾问+助攻工具人"

## 女性角色成长弧

- 女主角的成长必须有内外双线: 外在线（能力/地位/身份的提升）+内在线（认知变化/情感成熟/价值选择）
- 每次重大转折后需有≥1 章"内心消化"环节——女主角反思/复盘/确认新的认知——不能事件过了就翻篇
- 女主角的弱点需真实呈现——可以有性格缺陷/判断失误/能力不足——但不能损害读者对她的共情
- 女性配角需有独立于女主的成长线——不能所有女性角色的存在意义都与女主绑定
- "大女主"≠"玛丽苏": 强大的能力必须有合理的获取过程和对应的代价，万人迷属性必须有具体的人格魅力支撑

## 男性角色塑造

- 男性角色的魅力需通过**具体行为**展示: 能力/担当/温柔/幽默/克制——不能靠旁白说"他很强大很深情"
- 男主角的缺点需与优点同源: 温柔可能带有优柔寡断/强大可能带有霸道/深情可能带有占有欲——不能"所有优点集于一身无任何缺点"
- 男主的情感转变必须有触发事件: 从冷淡到在意/从公事公办到私心维护——每次推进需要≥1 个具体触发
- 男配角的塑造标准不低于男主: 有独立的性格/动机/行动线——不能只是"男主的替代品"或"痴情备胎"
- "霸总"人设细化要求: 霸道是手段不是本质——本质是保护欲/责任感/不安全感的扭曲表达。霸总不能等同于"无理由控制女主一切"

> 例：霸总男主→优点:果断、有手腕、在危机中保护女主→缺点同源:控制欲强、不善表达关心、用"命令"代替"请求"→情感转变触发:女主在他病倒时不求回报地照顾(Ch15)→他开始笨拙地学习"询问她的意见"而不是"替她做决定"(Ch16-18)

## 氛围与审美描写

- 重要情感场景需有环境氛围烘托: 光线/季节/气味/声音——环境与情感同频（悲伤=阴雨/黄昏/冷茶），但不能每场都同频（需要反差营造张力）
- 服饰/饮食/住所描写需有叙事功能: 展示身份/反映心境/暗示关系/埋设伏笔——不能为描写而描写
- 美的描写需具体而非堆砌形容词: "她穿着一件月白色的衫子，袖口的绣线被洗得有些发白——这件衫子穿了三年了" > "她穿着一件漂亮的白色衣服"
- 审美节制: 每章纯审美描写（无叙事推进）的篇幅≤15%，重要场景可放宽至 25%

## 节奏与情感起伏

- 情感温度需有起伏: 甜→微虐→甜→大虐→更甜——不能持续甜或持续虐超过 5 章
- 每章必须有≥1 个情感触点: 让读者产生共鸣/心疼/心动/愤怒/期待的瞬间
- 高潮场景前需≥3 章的情感积累——不能突然来一个高潮没有铺垫
- 虐点之后必须有糖的暗示: 即使是纯虐文，也需在虐的缝隙中留一线希望

## 常见错误

以下错误会导致女频读者流失或口碑崩塌，必须避免：

- **情感空洞化**: 写"她很感动"但不写怎么感动 → 每次情感描写需有内心反应链（事件→感受→身体反应→回忆→行为倾向）
- **女主工具化**: 女主的存在意义全靠与男主的关系定义 → 女主需有独立于感情的成长目标和独立行动线
- **男性角色扁平化**: 所有男性角色=不同类型的"完美男友模板" → 每个男性角色需有缺点和独立动机，缺点与优点同源
- **闺蜜功能化**: 闺蜜只用来当情感顾问/助攻/对比板 → 闺蜜需有独立的性格/目标/困境，不能只是女主的附属
- **虐而无望**: 连续虐超过 5 章无任何甜/希望的暗示 → 虐的缝隙中需有一线希望（即使微小），否则读者会弃文
- **场景描写装饰化**: 服饰/饮食/住所描写与叙事无关 → 审美描写需有叙事功能（展示身份/反映心境/暗示关系/埋设伏笔）

## 与其他 skill 的交互

- 本 skill 是**读者向元规则**——与任何题材 skill 叠加，不能取代题材 skill 的核心设定
- 与 romance 叠加时: 本 skill 的情感描写和男性角色塑造规则激活。romance 提供 CP 数量和感情线结构，本 skill 提供女性视角下的情感体验标准和男性角色魅力标准
- 与 palace 叠加时: 本 skill 的情感体验优先规则与 palace 的权力博弈融合——宫斗/宅斗中的情感关系受权力约束，但情感体验描写不可因权谋而被简化
- 与 female-oriented 叠加的 setting 类 skill（xianxia/wuxia/high-martial 等）: 战力/修炼体系照常遵守——但涉及女性角色的能力描写时，需同时注重"能力获得的代价和心路"（本 skill 要求）
- 本 skill 不定义世界观/战力/情感结构的具体设定——只定义叙事风格和角色塑造的读者向标准
