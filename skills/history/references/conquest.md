# 历史争霸 / 种田

诸侯争霸、统一战争、种田发育、根据地建设。核心：根据地经营、军事扩张节奏、人才招揽、经济基础与战争机器的平衡。

## 根据地经营

- 初始根据地的选择需有地理逻辑: 有险可守（山脉/河流）+有粮可食（产粮区）+有路可通（交通线）+有险可控（边境威胁可控）。完美根据地不存在——至少有一个维度是短板
- 根据地的发育阶段: 站稳脚跟（生存）→ 发展农业（温饱）→ 招揽人口（扩张基础）→ 建立手工业（装备自给）→ 吸引人才（管理团队）→ 向外扩张。每阶段需≥10章
- 内政建设需有优先级排序: 粮食安全>军事防御>人口增长>税收制度>文教发展>商业繁荣。不能"同时推进所有建设"
- 种田文的资源循环: 开垦→增产→人口增长→需要更多土地→扩张或内卷。这个循环是种田文的核心叙事引擎

> 例：主角占据一座边陲县城起家→第一年:修缮城墙+开垦荒田+收拢流民(人口从3000到8000)→第二年:建铁匠铺(自制农具+武器)+训练500乡勇→第三年:粮仓充实+手工业起步→第四年:试探性进攻邻县→每一步的时间跨度和资源积累需合理

## 军事扩张节奏

- 扩张阶梯: 一县→数县→一郡→数郡→一州→半壁天下→统一。每级的实力门槛（兵力/粮草/将领/政治合法性）需明确
- 扩张后的消化期: 占领新地盘后需要≥5章的内部整合——清剿残余/安抚民心/任免官员/征收赋税/防止反叛。消化完成前不能开启下一次扩张
- 统一速度的硬约束: 天下大乱到重新统一通常需 30-50 年（如三国/五胡十六国/五代十国）或 10-20 年（如隋末/元末）。主角可以加速但需有合理理由（敌方分裂/主角军事天才/收编而非消灭）
- 合纵连横: 多势力并存时，主角需有外交策略——联合次要敌人打击主要敌人/远交近攻/以弱制强联盟。不能靠纯军事碾压单挑全天下

## 人才招揽与任用

- 人才的来源需合理: 本土地主/外来流亡者/科举取士/降将降臣/民间访贤。每种来源的人才各有特点（忠诚度/能力/利益诉求不同）
- 文臣武将的梯队建设: 元老（起家时跟随）→ 中期（扩张中加入）→ 后期（统一前后归附）。不同梯队的利益冲突是天然的政治张力
- 降将降臣的处理: 接纳但不完全信任/需要投名状/给予地位但限制权力。降将反复的问题需≥5 章铺垫
- 人才的局限性: 早期能臣在统一后可能不适应治理——开国元勋≠治世能臣。人才的适用期和转型困难需展现

## 经济与战争机器的平衡

- 战争的经济账: 动员 10000 士兵 = 30000 人脱离生产（士兵 + 后勤），每天消耗的粮食 = 500 石。一场持续一个月的战役 = 动员人口中几个月到一年的劳动剩余
- 税赋与民力的平衡: 穷兵黩武→民力枯竭→流民起义→内部崩溃。军事扩张的规模和频率受经济基础限制——不能"一边打仗一边经济蓬勃发展"
- 以战养战可行但有限度: 攻占富庶地区可以补充军需，但破坏的战区反而成为财政负担。以战养战不能连续使用≥3 次
- 贸易在争霸中的角色: 控制关键贸易路线（盐/铁/马/粮）> 单纯扩大领土。经济封锁的战略价值需体现

## 合法性与人心

- 军事征服之外的政治合法性是必要条件: 禅让/前朝名义上的授权/血统宣称/"天命"符号。纯靠武力建立的政权极不稳定
- 民心不是口号而是具体指标: 粮价稳定/徭役合理/刑狱公正/基层小吏不鱼肉百姓。每次政策对民心的影响需≥3 章后展示
- 士绅和宗族的支持是基层统治的关键: 皇权不下县——县以下靠士绅和宗族。获得士绅支持=获得基层治理网络
- 统一后的大赦/减免赋税/表彰忠烈等"收心"举措是必经环节——不能统一了就万事大吉

## 常见错误

- **种田秒变暴兵**: 一个县三年爆十万兵 → 征兵比例上限≈人口的 8%-10%（极限动员），且大量征兵导致农业崩溃
- **名将批发市场**: 主角出门就遇到一堆未出山的名将排队来投 → 人才获取需合理来源和招揽过程，降将需完整转变弧
- **统一过速**: 十年统一天下 → 统一速度受消化期/后勤/合纵连横约束，合理的统一周期需根据初始实力设定
- **战争零成本**: 连年征战但经济越来越好 → 战争消耗有经济账，以战养战有限度，穷兵黩武会导致内部崩溃
