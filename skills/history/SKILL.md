# 历史写作规范

## 目标

确保历史类小说的世界观自洽——时代考据不出现硬伤（张冠李戴/跨时代技术出现）、
技术树与生产力水平匹配、社会结构符合历史发展阶段、军事制度与时代对应、
变革与改革有合理阻力、不出现"现代人穿越后降维碾压全体古人"的倾向。

## 建议创建的知识库文件

单独使用本 skill 时建议: 时间线.md（重大历史事件的精确时序）、势力追踪.md（诸侯/军阀/门阀/列国）、
地理.md（行政区划/地形/交通）、科技树.md（当前技术水平和研发进度）。
视剧情: 战力系统.md（有个人武勇/武将单挑时）、场景追踪.md（多都城/要塞时）、经济.md（有种田/贸易线时）。
人物卡: 启用阵营/所属势力字段。

## 时代考据约束

- 具体到朝代/年代设定的作品：核心制度（官制/兵制/税制/科举/法律）必须与设定朝代一致。不能明朝背景出现清朝制度
- 架空历史作品：需在 AGENTS.md 或大纲中明确架空基于哪个/哪些真实朝代（如"基于唐+宋的架空王朝"），核心制度以此为准
- 技术出现时间必须与时代对应: 不能汉朝出现火药武器/宋朝出现蒸汽机/明朝出现电报。架空历史中技术提前需≥15 章铺垫（技术引进/天才发明家+合理物质基础）
- 衣食住行的时代特征不能出错: 唐朝没有椅子（跪坐为主），宋朝以前没有大量白银流通（铜钱为主），明朝以前没有辣椒/红薯/玉米（美洲作物，明中后期传入）
- 称呼和称谓需符合时代: 不同朝代对官员/亲属/社会身份的称呼不同。如"大人"在宋元以前不用于称呼官员（用于称呼父亲），"老爷"在明代以后才普遍使用

> 例：不能犯的错误——写唐朝背景→主角吃辣椒炒肉（辣椒明朝才传入）+用白银付账（唐朝主要用铜钱和绢帛）+官员互称"大人"（唐朝称官职如"李尚书"或"明公"）→三个硬伤在一章内

## 技术树与生产力

- 科技树.md 中按时代建立技术清单: 已有技术（该时代的常规水平）/前沿技术（该时代顶尖但存在）/未来技术（主角引入但需≥15章铺垫）
- 技术升级需有产业链支撑: 一项新技术的落地需要原材料供应+工匠技能+市场需求+运输条件——不能主角画张图纸就量产
- 生产力的天花板由农业社会限定: 古代亩产极限（水稻约 200-300 斤/亩，小麦约 100-150 斤/亩）、运输成本（陆运≈粮食自耗 30%-50%/500 里）、人口上限（受耕地面积和亩产约束）
- 主角引入现代技术需有合理过程: 理论可行→找到工匠→反复试错→小规模成功→逐步推广。每阶段≥3章的合理过程

> 例：主角引入曲辕犁（唐朝已有的新技术）→找到铁匠打造→在自家田试用→亩产提升20%→邻居效仿→县令注意→推广到全县→这个过程需要≥10章而非1章

## 社会结构约束

- 古代社会的四大阶层需有对应角色和情节: 士（官员/士绅/读书人）/农（自耕农/佃农/流民）/工（工匠/匠户）/商（坐商/行商/海商）。不能所有角色都来自同一阶层
- 社会流动性必须符合时代: 科举制度下平民向上流动是可能的但极其困难（录取率约 1%-3%）；无科举的时代社会流动性更差
- 阶层冲突是天然的情节来源: 士绅与农民的租税矛盾/商人与士绅的地位不平等/工匠的社会地位问题/流民问题（古代社会最大的不稳定因素）
- 宗族/乡绅在基层治理中的角色不可忽视: 古代"皇权不下县"——县级以下的秩序由宗族/乡绅/里甲维护。主角的基层改革必须面对这一层

## 军事制度约束

- 兵制与时代对应: 征兵制（秦汉）/府兵制（隋唐前期）/募兵制（唐后期/宋）/卫所制（明）/八旗+绿营（清）。不同兵制决定了军队的战斗力/忠诚度/财政负担
- 兵种技术需合理: 骑兵是冷兵器时代的战略兵种——没有马场就没有骑兵（中原王朝的天然短板）。火器出现后（宋以后）战场形态渐变但冷兵器长期存在
- 军队后勤是硬约束: 古代"千里运粮"的消耗比（30%-50%自耗率）、行军速度（步兵日行 30-40 里，骑兵 60-80 里）、天气对军事行动的影响（雨季/冬季）
- 武将的个人武勇有限度: 可以在单挑中展现个人勇武，但不能一人杀穿一整支军队。武将的核心价值在指挥而非个人战力。可叠加 wuxia/high-martial skill 放宽武勇上限，但需明确标注

> 例：远征 3000 里需要 100 天行军（步兵日行 30 里）→ 5 万大军日耗粮 500 石 → 运输队自身消耗 + 路损 = 后方需发出 1000 石/天 → 需要 3000 辆运粮车 + 同等数量的民夫 → 这是古代战争的真实约束

## 变革与改革逻辑

- 任何改革都面临三股阻力: 既得利益者（旧制度下的受益者）/执行层的惰性（基层官员的阳奉阴违）/习惯的力量（被统治者的惯性）。至少展示两种阻力
- 改革的推进不是线性的: 尝试→遇到阻力→调整策略→部分成功→新问题出现→再次调整。成功的改革也需要≥15 章的曲折过程
- 激进改革（如废除科举/土地国有化/取消等级制度）在古代社会的失败率极高——可以有改革理想，但需有对应代价
- 改革的成果需要≥5 章后才能看到实际效果（政策传导延迟）

## 常见错误

以下错误会导致历史文失去厚重感和真实感，必须避免：

- **历史硬伤三连**: 朝代制度/技术/习俗三处以上出错 → 核心制度/技术/衣食住行需考据，架空历史需在 AGENTS.md 明确"基于哪个朝代"
- **技术火箭式跃进**: 主角三年内从农耕文明跃到工业革命 → 技术升级有产业链约束，每项新技术需≥15章合理过程
- **改革无阻力**: 主角推什么改革都顺利通过 → 任何改革面临≥2种阻力（既得利益者/基层惰性/习惯力量）
- **战争无后勤**: 十万大军说走就走，从不算粮草 → 军队后勤是硬约束（运输自耗率/行军速度/天气影响）
- **全员士绅化**: 所有角色都是官员/读书人/富商 → 古代社会四大阶层（士农工商）需有对应角色和情节
- **穿越者降维碾压**: 现代人穿古代轻松降维打击所有人 → 古代精英在自己的领域（权谋/军事/人心洞察）远超现代普通人，穿越者的优势在于知识而非智慧

## 与其他 skill 的交互

- 本 skill 定义历史时代的物质基础和社会运行规则。当与 transmigration 叠加时: transmigration 提供穿越者的身份适应和知识应用约束，本 skill 提供历史世界的运行规则——两者互补，不重叠
- 与 political（官场）叠加时: political 的古代官场 reference 与本 skill 兼容——political 聚焦权力体系内部运作，本 skill 提供更宏大的时代背景
- 与 palace（宫斗/宅斗）叠加时: 宫斗/宅斗发生在历史时代背景下——本 skill 的时代考据约束同时适用于后宫/内宅描写
- 与 wuxia/xianxia 叠加时: 武侠/仙侠发生在历史或架空历史背景中——本 skill 的时代考据约束（衣食住行/社会结构/制度）依然适用，但军事/技术约束可被战力体系部分覆盖
- 与 male-oriented 叠加时: 本 skill 的事业成长线（统一/建设/改革）天然符合男频叙事节奏；与 female-oriented 叠加时: 时代背景下的人物关系和情感体验是女频历史文的核心
