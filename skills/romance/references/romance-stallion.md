# 种马

核心是数量和征服感——女性角色快速加入，以性关系为主，不要求情感深度。与后宫的区别：后宫侧重"收集不同性格的女性并维持关系"，种马侧重"征服和性体验的多样性"。

## 角色类型多样化

- 每个新加入的女性角色必须有≥1 个与前人不同的**类型标签**：身份（女帝/师姐/魔女/邻家/寡妇/人妻）/体型/性格/与主角的相遇场景
- 同类型女性不能连续出现：连续 3 个"冷艳宗门天才"或连续 3 个"傲娇公主" → 类型重复。读者想要的是不同类型带来的新鲜感
- 角色的深度不是种马的重点——允许"出场→被征服→淡出"的模式。但淡出前至少要有 1 次完整的"相遇→征服"场景

## 情感约束的放松

- **情感推动约束（每章推进≥一条线、情感变化需触发事件）在种马文中大幅放宽**：
  - 允许"见色起意"——主角对女性角色的兴趣始于外貌/身材而非性格
  - 允许"一见倾心"——女性角色对主角的吸引力无需多章铺垫
  - 允许"一夜留情"——有些关系可以就停留在一夜，之后该角色可以不再出场
- 种马不需要后宫式的"关系管理"——女性角色之间不一定需要互动。她们围绕主角而不是互相围绕

## 场景多样化

- 征服场景（不仅是性，还有初次相遇、征服过程）需有变化：
  - 相遇方式轮替：英雄救美/实力碾压/智谋取胜/机缘巧合/对方主动
  - 征服场景的节奏：不能连续 3 个女性角色用同一种方式被征服
- 场景之间应有间隔章——不能连续 3 章每章只是"遇到新女人→征服→遇到下一个"

## 常见错误

- **类型单一化**: 所有女性角色都是同一种类型（如全部高冷师姐）→ 种马的核心卖点是多样性，类型重复是致命的
- **征服即消失**: 征服后的女性角色在后续章节完全消失 → 至少偶尔安排她们出场（如后宫聚会/修罗场/主角需要帮助时回忆某人的特殊能力）
- **种马装纯爱**: 明明是种马文却硬套纯爱的"情感专一"设定 → 类型定位要一致——不要一边收女人一边说主角只爱一个
