# 后宫

主角（通常为男性）与多个女性角色建立感情关系。女性角色围绕主角展开——这是男性向后宫的核心设定。约束重点在于**角色差异化**和**后宫内部关系**，而非角色的独立性。

## CP 数量与新人加入

- CP 数量无硬性上限，但每新增≥3 个后宫成员后，必须有≥1 章展示后宫内部互动
- 新成员加入节奏：不能连续 3 章每章新增一个——需要间隔章来展示已有成员的出场和互动
- 每个后宫成员首次出场时必须有**区别于已有成员的特征**：不同的性格/身份/能力/与主角的相遇方式——不能全是"冷艳师姐"的复制品

## 角色差异化

- 后宫成员之间必须有可辨识的差异：≥2 个维度上各不相同（性格/身份/能力/体型/口癖/与主角的互动模式）
- 每个成员在后宫中承担不同的"角色"：如智囊型/战力型/治愈型/傲娇型/温柔型——分工避免功能重叠
- 成员的差异化特征在加入后宫后仍需保持——不能加入后性格就变成"温柔贤惠"的统一模板
- 角色的背景故事（为什么对主角死心塌地）需≥1 章交代——不能全员都是"被主角实力折服"这一个理由

## 后宫关系管理

- 后宫内部的关系必须多样化——不能所有成员都"和平共处、其乐融融"
- 必须存在的关系类型（至少出现 2 种）：竞争（争夺主角关注）、盟友（性格互补互相支持）、联盟（两个弱势成员联手对抗强势成员）、冷淡（不关心其他人只在意主角）
- 后宫和谐的推进需要过程——不能三章前还在互相敌视三章后就成闺蜜
- _关系与称呼索引.md 中需记录成员之间的关系——不仅是"每个人↔主角"，还有"成员↔成员"

## 常见错误

- **同质化后宫**: 所有成员性格/身份/外貌趋同 → 每个成员必须有≥1 个独特标识，且加入后该标识仍需保持
- **征服即消失**: 女性角色被征服后不再出场 → 已加入的成员需在后续章节中至少保持可见度，不能沦为背景板
- **后宫全和谐**: 所有成员加入后立刻和平共处无矛盾 → 后宫关系需要经营，至少存在 2 种以上的关系动态
