# 黑科技写作规范

## 目标

确保黑科技小说的技术叙事有可信度——技术研发有合理工程阶梯而非"灵光一闪就造出成品"、
专利与商业竞争逻辑自洽、外部势力（政府/军方/资本）的关注有合理反应时间与应对方式、
技术的社会影响有展示、不出现"一项技术单挑全世界"的降维碾压幻想。

**与硬科幻的区别:** 黑科技文允许比硬科幻更宽松的科学合理性——可以有一两处"超过当前科技树"的设定，但设定一旦确定就必须内部自洽，不能无限追加新能力。黑科技=一项或几项"奇迹技术"+真实商业/政治/社会逻辑。

## 技术研发约束

- 每项核心技术的研发需有完整的工程阶梯: 理论验证→实验室原型→小型试产→中试放大→量产。每阶段≥5章的合理过程。不能"主角有了一个想法→三个月后产品上市"
- 技术的"边界条件"首次出场时完整声明: 原理（大致方向，可略过数学细节）/应用范围/性能上限/技术瓶颈/依赖的关键材料或工艺。声明后不可追加新功能（除非通过版本迭代——见下）
- 技术迭代需有合理节奏: 1.0版（基础功能）→1.1版（修复缺陷）→2.0版（重大升级）→3.0版（革命性突破）。版本之间的间隔需≥15章（1.0→1.1等小迭代可缩短至5章）。每次迭代需展示技术瓶颈的突破过程
- 失败是研发的一部分: 至少每20章展示1次研发失败（实验失败/原型机无法达标/量产良率太低/被竞争对手抢先）。失败→分析原因→改进→重试——这才是真实的技术研发

> 例：主角开发新型电池→理论验证(利用石墨烯+特殊电解液，密度可达锂电池3倍)→实验室原型(做出纽扣大小的样品，充电1次点亮LED3天)→小型试产(做出手机电池大小的成品，良率仅30%+成本太高)→中试放大(建中试线，良率提升到70%，成本降至锂电池2倍)→量产(良率95%+成本降至锂电池1.5倍)→市场接受(价格贵但续航翻倍)→竞争对手反制(传统电池厂降价+抹黑安全性)→这是一条合理的科技创业之路

## 专利与商业竞争

- 专利布局需有策略: 从研发初期就开始申请核心专利→产品上市前完成专利围墙（核心专利+外围专利+外观专利+国际PCT）→上市后应对侵权行为。不能"产品已经卖了才发现忘了申请专利"
- 竞争对手的反应需合理: 技术领先→小公司抄袭（可诉讼制止）→中公司开发替代技术（需时间应对）→大公司利用渠道/资本/政策打压（最难的阶段）→行业联盟制定标准（主角参与标准制定=终极胜利）
- 商业竞争的手段多样: 价格战/专利诉讼/挖人/抹黑PR/上下游封锁/标准战争。不能只靠"技术更好"就赢得市场
- 科技创业的资本路线: 自有资金→天使轮→A轮→B轮→IPO。每轮融资对应不同的估值/股权稀释/对赌协议。资本不是白拿的——投资方有自己的利益诉求

## 外部势力的反应逻辑

- 政府对颠覆性技术的关注有合理反应时间: 技术报道→行业关注(1-3个月)→政府部门调研(3-6个月)→政策调整/安全审查/军方接触(6-12个月)。不能"今天产品上线明天国安就上门"
- 军方的兴趣是双刃剑: 好处是获得资金+资源+背书——代价是技术受限/出国限制/保密义务/技术被军方控制。军转民/民转军的技术转让有现实的法律和政策壁垒
- 外国势力的觊觎需有合理手段: 情报收集（公开信息+商业间谍）→技术合作（合资/授权/引进）→技术封锁（禁运/切断供应链）→人员挖角→法律战。不能"外国人想偷所以派特工直接来抢"
- 技术的社会影响: 技术（如自动化/AI/新能源）对就业/产业结构/社会生活的影响需展示。主角不能只享受技术带来的成功而不面对技术带来的社会代价（传统行业从业者失业/伦理争议/技术依赖风险）

## 团队与人才

- 核心技术团队的构建: 主角（技术愿景+核心能力）+技术合伙人（互补技能）+商业合伙人（管理/市场/融资）——三驾马车缺一不可。不能主角一个人搞定所有
- 人才招募有竞争: 顶尖人才被大公司/国家实验室争抢——主角凭什么吸引他们？（技术愿景/股权激励/科研自由/使命感）
- 核心人员的离职/背叛是最沉重的打击: 带走技术/带走团队/卖给竞争对手。防范措施（竞业协议/技术分段隔离/专利保护）需提前布局
- 创始团队的股权分配和内部矛盾是天然的戏剧张力: 技术贡献vs资金贡献/早期加入vs后期加入/不同方向的分歧。初创公司比大公司更脆弱——内部矛盾的影响力更大

## 常见错误

以下错误会导致黑科技文沦为"技术意淫"而失去可信度，必须避免：

- **研发零成本**: 主角有想法→直接造出完美产品→跳过所有研发阶段 → 每项技术需完整工程阶梯（原型→试产→中试→量产），每阶段≥5章
- **技术无边界**: 黑科技初期设定是"新型电池"，后期变成了"万能能源+武器+航天发动机" → 技术边界首次出场完整声明，版本迭代有合理过程
- **商业真空**: 主角技术无敌→全行业立刻接受→无人竞争 → 竞争对手的反应分级（抄袭→替代→打压→联盟），至少存在2种竞争手段
- **政府反应过激**: 刚做一个原型→国安/军方/外国特工全来了 → 外部势力关注有合理反应时间（月到年级），手段在合理范围内
- **主角全能**: 技术研发+商业谈判+法律诉讼+政府公关+人员管理全是主角一个人干 → 团队构成需合理（技术+商业+管理），术业有专攻
- **技术无代价**: 没有任何技术失败/安全问题/社会争议 → 研发有失败（每20章1次），新技术有安全隐患，社会影响有争议

## 与其他 skill 的交互

- 本 skill 定义技术研发的商业化和社会化逻辑。与 scifi 叠加时: scifi 提供更宏大的世界观和未来设定，本 skill 提供技术落地的工程/商业/社会约束
- 与 history 叠加时: 穿越历史+黑科技→现代人在古代/近代引入超前技术。技术与时代生产力的匹配约束由 history skill 提供，本 skill 提供技术研发的工程阶梯
- 与政治 skill 叠加时: 科技企业与政府的互动（政策支持/安全审查/军方合作）有官场逻辑——political skill 的权力体系约束可在这一层面叠加
- 与 power-fantasy 叠加时: 爽文节奏可放松部分约束（竞争对手的强度可降低、融资速度可加快、政府审批可简化），但技术研发的工程阶梯不可跳过、专利与商业的基本逻辑不可省略
- 科技树.md 是本 skill 的核心知识库文件——记录每项核心技术的研发阶段/性能参数/技术瓶颈
