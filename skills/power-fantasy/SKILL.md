# 爽文写作规范

## 目标

确保爽文的情绪节奏满足读者期待——通过主角实力碾压、打脸反派、逆袭翻盘等情节
制造阅读快感。当爽文规则与其他 skill 约束冲突时，爽文规则优先（但有明确底线）。

## 建议创建的知识库文件

与具体流派 skill 叠加使用，无额外文件需求。

## 核心原则: 爽的优先级与底线

当以下约束与爽感冲突时，可以放松:
- 其他 skill 的战斗平衡约束（主角可跨级碾压，前提是展示≥1 个碾压手段）
- 资源获取约束（奇遇/天降机缘的频率可提高，但每次需≥1 句合理来源简述）
- 角色反应约束（配角的震惊/嫉妒/悔恨可以放大，但同一角色同类反应≤3 次）
- 时间流速约束（修炼/建设速度可加快，但不能跳过大阶段）

不可放松的底线:
- 世界观核心规则不能矛盾（如果设定说"没有复活术"，不能为了爽让死人复活）
- 人物性格不能为爽崩坏（反派可以蠢但不能突然变蠢）
- 已设定的能力限制不能被无视（如果主角的剑是火属性，不能突然用来灭火）
- 因果关系不能断裂（事件 A→事件 B 的链路必须有，只是可以简化）

> 例：修仙爽文中，主角筑基期打败金丹期反派——允许(展示了血脉觉醒+阵法陷阱两个碾压手段)，但金丹期的基本能力(灵力化形)不能被无视

## 打脸节奏约束

- 每≤3 章至少一次"打脸"事件
- 打脸类型需轮替: 实力碾压/身份揭晓/智谋反转/背景压制/众目睽睽下的翻盘
- 同一反派被连续打脸≤2 次——之后需引入更强反派或该反派退场
- 打脸前必须有铺垫（反派先嚣张/轻视/威胁），铺垫和打脸之间的间隔≤2 章
- 每次打脸后至少 1 句"围观者反应"描写来放大爽感

## 反派管理约束

- 反派梯度: 新手村反派→地方势力→国家级→大陆级→世界级→跨世界级
- 每梯度至少 2 个反派，不同性格/手段——不能全是"看不起主角的纨绔"
- 反派退场方式需多样: 被消灭/被收服/被迫合作/逃跑后升级归来
- 反派不能降智: 反派的行为逻辑必须自洽——只是信息不足或性格自负导致误判

## 节奏约束

- 每章必须有≥1 个爽点（小爽即可）: 主角获得新能力/新道具/新信息/新崇拜者
- 每 5 章必须有≥1 个大爽点: 打脸关键反派/境界突破/获得重大机缘/身份揭晓
- 连续爽→情绪麻木: 每 3 个大爽点后需插入≥1 章的缓冲（日常/准备/铺垫新反派）
- 爽点密度不能持续上升——需要起伏。低谷和高峰交替

## 机缘管理约束

- 机缘≠天上掉馅饼: 每次机缘必须有≥1 句来源解释
- 机缘的价值需与当前实力成正比——筑基期获得金丹传承=合理，获得神王传承=过度
- 机缘的副作用或隐藏风险至少暗示一次（即使主角不知道）
- 同类型机缘（如探索遗迹→获得功法）不能连续使用≥3 次

## 常见错误

以下错误会导致爽感疲劳或世界观崩坏，必须避免：

- **爽感疲劳**: 连续 5 章打脸无缓冲 → 每 3 个大爽点后插入≥1 章缓冲
- **反派复制粘贴**: 所有反派都是"看不起主角的纨绔"→ 每梯度至少 2 个反派且性格/手段不同
- **机缘无代价**: 主角无条件获得顶级机缘 → 每次机缘需≥1 句来源解释，副作用至少暗示一次
- **为爽崩人设**: 冷静型主角突然无理由大开杀戒 → 人物性格变化需有触发事件

## 与其他 skill 的交互

- 本 skill 的爽感优先级高于部分逻辑约束——当本 skill 激活时，其他 skill 的以下约束可适度放松: 战斗平衡、资源获取频率、角色反应的强度、时间流速
- 不可放松的底线（不可被本 skill 覆盖）: 核心规则一致性、人物性格不崩坏、已设能力限制不无视、因果关系不断裂
- 本 skill 的节奏约束（打脸≤3章/反派梯度/机缘管理）独立于其他 skill——即使叠加也遵守此节奏
- 当与 romance 叠加时: 打脸节奏优先于情感细水长流的节奏，但情感转折不可为爽牺牲
