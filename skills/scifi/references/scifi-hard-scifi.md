# 硬科幻

以已知科学为基础、严格推演技术可能性——不引入"黑科技"或未解释的魔法式科技。

## 科学一致性

- 所有核心技术必须有**明确的理论基础**——不需要引用真实论文，但世界观内的科学逻辑必须自洽
- 科学设定一旦确立不可在后续章节为了剧情方便而违背
- 如果引入突破当前物理学认知的技术（如超光速），必须给出世界观内的理论解释——哪怕是一句话的"引力场方程的特殊解"
- **禁止"魔法式科技"** ——不能主角按一个按钮就解决所有问题而不解释原理

## 技术推演

- 技术发展的速度和路径需合理——从实验室验证→工程原型→量产→普及，每个阶段有时间间隔
- 新技术的副作用和限制必须在首次出场时至少暗示——不能需要一个完美解决方案时就"恰好"出现一个无副作用的新技术
- 产业链和基础工业对技术实现的影响需考虑——不能在一个刚恢复农业的殖民地突然造出反物质引擎
- **人类生理限制**：即使技术解决了外部问题（如温度/辐射），人体自身的耐受极限（加速G力/长期失重/宇宙射线累积）仍然是硬约束

## 与太空歌剧的区别

- 硬科幻可能**没有**FTL、外星文明、能量护盾——甚至可能只局限在太阳系内
- 如果设定中包含外星生命，需从科学角度解释：基于什么生物化学？进化路径？星球环境的约束？
- 硬科幻的"冲突"可以是技术问题本身——工程师努力解决一个看似不可解的技术难题，而非用武力战胜敌人
