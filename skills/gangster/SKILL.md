# 黑道写作规范

## 目标

确保黑道/地下世界观内部自洽——帮派权力体系不混乱、地盘争夺有因果链、
暴力行为有代价与后果、黑白两道互动关系可追溯、忠诚与背叛有完整动机链。

## 建议创建的知识库文件

单独使用本 skill 时建议: 势力追踪.md（帮派/堂口/敌对势力）、场景追踪.md（地盘/产业场所）。
_关系与称呼索引.md（帮派内称谓、黑白两道关系）。
视剧情: 道具追踪.md（关键信物/武器/账本）、时间线.md（重大事件的时序）。

## 帮派权力体系

- 帮派等级结构在势力追踪.md 中明确定义: 龙头→堂主→头目→底层成员。每级标注权力范围、收入来源、直属人数
- 权力交接必须有因果链: 上位方式（继承/火并/禅让/外部扶植）→上位事件→上位后≥3 章展示权力稳固过程
- 帮派内派系需记录: 每个派系的利益诉求、关键人物、与外部势力的联系
- 等级变动记录在人物卡修为演变日志: 旧地位→新地位→触发事件→影响范围

> 例：阿坤从"庙街头目"升为"铜锣湾堂主"→触发事件:前堂主在火并中被杀+阿坤带人守住地盘→影响:手下从10人扩至50人+月入从5万增至30万+仇家从街头混混变为对头帮派堂主级

## 地盘与经济约束

- 地盘边界在场景追踪.md 中记录: 范围/产业类型/月收益/守护力量/争夺对手
- 地盘扩张必须有代价: 每获得一条街/一个产业需付出（人员伤亡/警方关注度提升/盟友利益让渡）
- 经济来源需可追溯: 保护费/走私/赌场/夜场/白道产业——至少 2 条经济线并行，单一线索不能支撑整个帮派
- 地盘丢失→经济萎缩→帮派实力下降的反馈链需≥2 章展示——不能无后果

## 暴力行为的代价

- 每次暴力行为（斗殴/火并/暗杀）必须产生≥1 种后果: 警方调查升级/对方报复/内部伤亡动摇人心/地盘不稳
- 警方关注度作为隐藏压力值: 暴力事件累积→警方压力上升→≥5 次暴力事件后必须有警方打击
- 主要角色死亡需有≥5 章铺垫，不能为爽感随意杀人（反派的死亡也需有合理落幕）
- 暴力升级需合逻辑: 街头斗殴→持械冲突→枪械→爆炸。跳级必须有外部势力介入或重大事件触发

> 例：小规模街头冲突(Ch3)→对方报复烧店(Ch5)→警方介入抓捕双方头目(Ch7)→帮派被迫低调3章→新保护伞到位后重新活跃(Ch11)

## 忠诚与背叛

- 忠诚建立需具体触发事件: 救命之恩/利益绑定/共同敌人/血缘关系——不能"莫名其妙就是兄弟"
- 背叛动机链必须完整: 外部诱惑（更高利益/对方胁迫）+内部不满（被轻视/分配不公/旧怨）+触发事件（最后一根稻草）
- 背叛发生后≥2 章展示连锁反应: 信息泄露→地盘失守→盟友动摇→内部清洗
- 忠诚测试不能滥用: 同一角色被测试≥3 次后需有明确结论（信任或清除），不能反复试探

## 常见错误

以下错误会导致黑道沦为儿戏或美化犯罪，必须避免：

- **暴力无代价**: 主角方随意杀人无后果 → 每次暴力行为需≥1 种代价（法律/社会/内部/心理）
- **警方真空化**: 大量犯罪活动发生但警方永远不出现 → 暴力事件累积→警方压力上升，≥5 次后必须介入
- **背叛动机空洞**: 角色突然背叛"因为他就是坏人" → 背叛必须有完整动机链（外部诱惑+内部不满+触发事件）
- **地盘扩张无成本**: 主角轻松吞并对家地盘 → 每次扩张需付出人员伤亡/资源消耗/外部压力
- **帮派内部无矛盾**: 所有人对龙头忠心耿耿 → 帮派内部必须有权力/利益/路线的张力

## 与其他 skill 的交互

- 本 skill 定义的暴力代价、权力体系和忠诚规则是黑道题材的基石。当与 power-fantasy 叠加时: 爽感可以放松战斗平衡（主角可以更强），但暴力代价不可取消——警方/社会/内部反应必须存在
- 与 romance 叠加时: 帮派内情感关系受权力等级约束（龙头的情人≠普通成员的女友），身份差异是情感障碍的来源
- 本 skill 的无警方/无政府状态约束与官场 skill（political）互斥——黑道以地下秩序为中心，官场以体制内权力为中心，不可叠加。如混合元素（黑道+官场勾结），选择占比≥70%的一方作为主 skill，另一方作为配角线索
- 当与 rebirth 叠加时: 重生者利用前世记忆获取先机合理——但帮派的等级结构、暴力代价、警方压力不可因重生而消失
