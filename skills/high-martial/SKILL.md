# 高武写作规范

## 目标

确保高武世界观内部自洽——武道体系上限在超人范畴而不滑向修仙、气血/基因/科技
强化各有代价与边界、战力排名有明确阶梯和晋升条件、竞技规则前后一致。

## 建议创建的知识库文件

单独使用本 skill 时建议: 战力系统.md（武道等级/战力榜）、功法技能.md（武技/秘法/基因能力）。
视剧情: 势力追踪.md（学院/武馆/国家战力机构）、场景追踪.md、道具追踪.md（武器/基因药剂）。

## 与仙侠/武侠的关键区别

- 无修仙元素: 无长生、无飞升、无灵气修炼、无金丹元婴
- 战力上限: 超人范畴。顶尖武者可碎楼破甲/音速移动/断金裂石，但不能移山填海/御空飞行
- 力量来源: 气血修炼/基因强化/科技改造/血脉觉醒——非灵力吞吐
- 时代背景: 通常为现代或近未来。科技与武道并行，而非古代江湖
- 战力体系需有科学化/系统化包装: 等级可量化（如F→SSS级），而非"意境""道心"等哲学化描述

> 例：高武 S 级武者全力一击可摧毁一栋五层建筑、时速可达200km、能硬抗步枪子弹——但不能飞行、不能活过150岁、不能用法术隔空取物

## 武道体系约束

- 等级阶梯在战力系统.md 中明确定义: 每级标注力量上限/速度上限/防御上限/晋升条件
- 晋升必须有量化测试或实战验证——不能"一朝顿悟连升三级"
- 同级内战: 胜负取决于武技克制/战斗经验/临场战术/装备差距
- 跨级挑战: 越一级需≥1 个优势（武技克制/装备碾压/战术压制），越两级需≥2 个优势+代价（重伤/透支），禁止越三级
- 等级晋升后在≥2 章内展示磨合期: 新力量不适应/被高级别盯上/旧敌升级

## 气血/强化体系约束

- 每种力量来源在战力系统.md 中标注: 获取方式/上限/副作用/与其他体系的冲突
- 基因强化: 每次改造有成功率(≥60%)、排斥反应期(≥2章)、基因崩溃风险
- 科技装备: 有能耗/维护成本/技术门槛——不能人手一件顶级装备
- 血脉觉醒: 需触发条件（濒死/极度愤怒/特定环境），觉醒后有冷却期≥10章，且觉醒≠无敌（同血脉仍分等级）
- 多种强化叠加: 需记录兼容性——气血+基因可行但排斥反应加倍，气血+科技可行但能耗更高

> 例：主角基因改造→成功率72%→成功提升至B级力量+手臂异化→排斥反应:高烧3天+战力暂时下降至C级→3章后稳定在新等级+异化手臂需学习控制(2章适应期)

**通用竞技规则（所有子类型适用）:**
- 每场战斗必须推进≥1 条线: 排名变动/角色成长/势力格局变化/信息揭示
- 竞技规则设定后禁止临时更改（除非作为情节事件: 主办方黑幕）
- 战斗中展示的新能力必须在之前≥3 章有训练/强化的铺垫

## 势力与国家结构

- 势力层级: 地方武馆→城市战力机构→省级→国家级→国际联盟。每级有明确的管辖权和战力门槛
- 战力榜/排名体系需定期更新（如每 10 章一次排名变动），排名变动触发竞争者反应
- 国家战力机构（如特殊部队/超能管理局）与民间武馆的关系需定义: 合作/竞争/管制
- 势力的资源（训练设施/基因药剂/武技秘籍）获取需有来源和竞争——最好资源集中在最高势力

## 常见错误

以下错误会导致高武沦为低配修仙或战力崩坏，必须避免：

- **高武修仙化**: 武者飞天遁地/活过500岁/修炼意境→境界树需上限明确，顶级≤超人范畴，寿命≤150岁
- **战力排行装饰化**: 战力榜设定后从不更新或角色等级与榜无关 → 战力榜每10章至少一次变动，排名变动触发情节
- **强化无代价**: 基因改造/血脉觉醒后立刻完美控制 → 每次强化后有≥2章的适应期/副作用/代价
- **学院万能化**: 学院内能学到所有顶级武技 → 最强武技/秘法需额外来源（遗迹/师承/机遇），学院只提供基础和中级
- **跨级无压力**: 主角轻松越级打败高阶 → 越一级需≥1个优势，越两级需≥2个优势+代价，禁止越三级

## 与其他 skill 的交互

- 本 skill 与 xianxia 互斥——不可叠加。高武的力量体系上限在超人范畴，不可混入修仙/飞升/长生元素
- 本 skill 与 wuxia 兼容但不建议——wuxia 的上限低于 high-martial，混合时战力体系以上限高的一方为准。如用户明确要"现代都市武侠"则使用 wuxia；"现代高武"则使用本 skill
- 与 scifi 叠加时: 高武+赛博朋克/基因改造的科技元素可深度融合，但力量体系的量化/等级规则不可弱化
- 与 power-fantasy 叠加时: 爽感可放松跨级限制（允许越级更轻松），但等级体系本身不可简化、强化代价不可取消
