# 灵异写作规范

## 目标

确保灵异/恐怖的超自然规则体系自洽——同类实体能力限制一旦设定不可改变、
恐怖氛围强度只能递增或维持、超自然知识获取有来源。

## 建议创建的知识库文件

单独使用本 skill 时建议: 超自然规则.md（实体行为规则/能力边界）。
人物卡: category=spirit 的角色。场景追踪.md（恐怖氛围感官描写）。

## 超自然规则约束

- 每种超自然实体的能力边界必须在首次出场时完整设定
- 规则不能前后矛盾——同一类实体的能力限制一旦设定不可改变
- 实体的规则在首次声明后不可为了剧情方便而随意追加

## 恐怖氛围约束

- 场景追踪.md 中记录每次恐怖场景的感官描写要素（光/声/温度/气味/触感）
- 恐怖氛围强度只能递增或维持，不能无故递减
- 连续≥3章的恐怖场景必须插入≥1章的缓冲（避免读者麻木）
- 恐怖触发条件需设定: 时间（如午夜）/地点（如特定房间）/行为（如念咒）

## 信息边界约束

- 超自然知识的获得必须有来源: 古籍/老人/亲身经历/灵异媒介
- 角色不能对超自然现象"天然免疫"——除非有设定支撑
- 目击超自然现象的角色心理反应需描写（恐惧/否认/崩溃/接受），不能跳过

## 常见错误

以下错误会导致灵异失去恐怖感，必须避免：

- **实体能力随意追加**: 实体第一章不能穿墙，第五章突然能穿墙 → 能力边界首次出场时声明后不可追加
- **恐怖递减**: 同类灵异事件多次发生后角色和读者都麻木 → 每次事件必须有新要素（更高级别实体/新能力/更个人化的威胁）
- **角色智商下线**: 角色的恐惧反应可以影响判断，但行为必须有性格逻辑支撑——不能为了制造恐怖场景而让人物做出不合常理的决策

## 与其他 skill 的交互

- 本 skill 定义超自然实体的行为规则和恐怖氛围管理。超自然规则独立于世界观物理/社会规则
- 与 mystery 叠加时: 探案逻辑 + 超自然变量 = 案件推理需同时考虑常规线索和超自然线索
- 超自然规则日志独立于战力系统——鬼怪的能力不是"战斗力"，是"规则"（限定条件下触发/被抵消）
