# 规则怪谈

规则是唯一的生存法则——发现规则、遵守规则、利用规则。常与无限流（副本化规则空间）、国运流（全球规则挑战）绑定。

## 规则发现

- 规则必须以**不完整的形式**逐步揭示——初次进入场景时只知道部分规则（通常从危险遭遇中推断）
- 规则的信息来源需合理：幸存者笔记/现场痕迹/反复试错/偶然触发。不能主角"推理"出从未出现过的规则
- 每条规则一旦被证实有效，就必须在全部场景中一致适用——除非场景本身有规则层级差异
- 禁止完整规则突然被全部揭示——规则必须是逐步拼图的过程

## 规则约束

- 每条规则必须明确定义：**适用范围**（某个房间/某个时间段/某个行为触发）和**违反后果**（死亡/失去身体部位/精神污染/驱逐出场景）
- 规则之间可以有层级——基础规则（如"天黑后不能出声"）和高级规则（如"血月之夜所有基础规则反转"）
- 规则的例外必须由更高层级的规则定义——不能无理由地突然例外
- 规则的"破解"不等于规则的"取消"——主角找到的是在规则范围内生存的方法，而非让规则失效

> 例：废弃医院规则 → 规则 1:天黑后不能在三楼停留(违反:被无形力量推下楼梯致死)/规则 2:听到婴儿哭声必须立刻闭眼数到 10(违反:睁眼看到不该看的东西，永久精神损伤)/规则 3(4 章后揭示):如果在三楼听到哭声，规则 2 失效，取而代之:必须在哭声停止前找到哭声源头并说出"你已经死了"

## 与无限流/国运流联动

规则怪谈天然适合作为无限流的副本或国运流的挑战关卡：

- **无限流联动**：每个副本是一个独立的规则空间。进入前只知道部分规则（或完全不知道），在副本中逐步发现规则。成功=理解规则后存活或通关。若叠加无限流，同时加载 [infinite skill](../../../infinite/SKILL.md) 的对应 reference
- **国运联动**：规则怪谈作为国运挑战类型。各国代表在同一规则空间中竞争——谁先发现隐藏规则谁获得优势。若叠加国运流，同时加载 [national-fate skill](../../../national-fate/SKILL.md) 的对应 reference
