## 多章 task

task 含多个章节路径时：**逐章**执行以下审计维度，最终报告**按章号分组**呈现。跨章模式时可额外检查：章间伏笔节奏是否均匀（每章新埋/推进/回收数量不应集中于某章）、人物弧线是否连续（无跳变无停滞）、因果边是否跨章闭合（无断头边）、POV 轮换是否均匀（同一 POV 连续 ≤4 章）。

batch ≥5 章时，报告中增加 `## 跨章节奏总览` 节：汇总各章字数、伏笔操作数、出场人物数的分布，标注集中/稀疏的异常章。

## 读盘顺序（硬性）

**第一步：结构化数据采集（同批并行）**

| # | 工具 | 参数 | 获取内容 |
|---|------|------|---------|
| 1 | `TrackingQuery` × 5 | file=scene/prop/faction/timeline/power, operation=current | 各类追踪表末行（确认已更新到本章） |
| 2 | `RelationQuery` | character=细纲出场人物名（逐一调用） | 每个出场人物的当前关系与称呼 |
| 3 | `ForeshadowTracker` | current_chapter="ChN" | 待回收伏笔状态（urgent/overdue/upcoming），关注密度 |
| 4 | `CharacterSearch` | query=出场人物名 | 定位各人物卡路径 |
| 5 | `PlotGraph` | traverse=backward/forward | 本章涉及事件的因果链路，检查闭合性 |
| 6 | `Corkboard` | 最近 5-10 章 | 场景卡片，检查人物轮换与 POV 分布 |
| 7 | `Stats` | chapter=最近已完成章 | 比对字数趋势 |

**第二步：补充精读（仅当结构化数据不足时）**

- 细纲：`Read knowledge/plot/细纲/chapter-NNN-细纲.md`（全文，通常 ≤60 行）
- 大纲当前卷概要：Grep `## 第X卷`（定位卷标题）→ Read 卷概要段 → 对比细纲核心事件
- 人物卡演变日志末行：`Grep "^\| Ch"` 定位 → Read ±2 行
- 大纲章节索引：Grep `"\| N \|"` 在对应卷内定位本章行 → Read ±1 行确认

## 审计维度

### 一、大纲对齐

细纲的核心事件是否与本卷大纲概要一致？

1. Read 细纲「场景拆分」汇总 → 提取本章核心事件
2. Read 大纲当前卷 `### 本卷概要` → 确认本章在卷级剧情中的位置
3. 对比：细纲的事件是否推动卷主线？是否有偏离卷级方向的支线？
4. 大纲章节索引中本章行「核心事件」是否准确反映细纲内容？

### 二、伏笔密度

1. 从 ForeshadowTracker 获取 urgent/overdue/upcoming 列表
2. 细纲「伏笔清单」中本章操作（新埋/推进/回收）各多少？
3. 判断标准：
   - 本章新埋 ≥3 且 overdue ≥2 → **伏笔堆积**，建议分摊
   - urgent 中有本章应回收但未列入细纲的 → **遗漏**
   - 预计回收章 ≤ 当前章+2 的伏笔集中在同一章 → **回收过于密集**

### 三、因果闭合

1. PlotGraph backward：本章核心事件的**前因**是否在之前的因果链中有记录？
2. PlotGraph forward：本章事件的**后果**是否有预期的后续边（至少有一条去向）？
3. 细纲「知识库更新清单」中因果链已打勾 → 验证因果链.md 中本章计划边是否合理
4. 断头边（孤立事件，既无前因也无后果）→ 标注

### 四、人物轮换

1. Corkboard 获取最近 5-10 章场景卡片，统计 POV 分布和出场频次
2. CharacterSearch 确认重要角色（主角、主要配角）最近出场章号
3. 判断标准：
   - 重要角色连续 ≥5 章未出场 → **轮换告警**
   - 同一 POV 连续 ≥4 章 → 建议切换视角
   - 单章出场人物 >8 人 → **人物过载**，建议拆分

### 五、字数分配

1. 细纲各场景 `~字` 合计 → 是否在 2000–4000 目标范围内？
2. 场景数是否合理（通常 3–5 个场景/章）？
3. 单个场景预估 >1500 字 → 建议拆分场景
4. 所有场景预估 <400 字 → 场景过多，建议合并

### 六、知识库登记完整性

1. 细纲「知识库更新清单」是否全部打勾 ✓？
2. 各追踪表（TrackingQuery 返回的末行）章节号是否 ≥ 本章？
3. 大纲章节索引是否有本章行？
4. 人物卡演变日志末行章节号是否 ≥ 本章（针对有变化的角色）？
